    法說會與重大訊息¶

    本篇你會學到¶

    • 法說會在說什麼
    • 如何抓重點、警惕話術與籌碼背離

    法說會是什麼¶

    法人說明會：上市公司向法人與分析師說明營運、財務與展望的會議，內容常公布於公開資訊觀測站（MOPS）。

    類型 內容
    業績說明 季報後最常見
    產品/產能 擴產、新產品進度
    展望 明年成長、毛利率指引

    閱讀重點清單¶

    1. 量化指引：營收、毛利率、CapEx 是否具體？
    2. 與前期差異：展望上修還是下修？
    3. 風險因子：管理層主動提的逆風是什麼？
    4. Q&A：分析師追問的尖銳問題與回答是否迴避？

    重大訊息¶

    上市櫃及興櫃公司一旦發生足以影響股東權益或股價之重大事件（併購、高層異動、財報自結大幅超預期、重大訴訟、天災等），須於第一時間在 MOPS 發布。

    類型 範例
    利好 接獲大單、擴產、配息
    利空 下修展望、訴訟、停工
    中性 人事、例行公告

    即時查詢：MOPS 重大訊息，可輸入公司代號與年月。

    事件驅動查證¶

    市場的極端報酬常來自未被預期的意外。把重大訊息與技術線圖疊看，能還原股價跳空當下的真實資訊環境：

    flowchart LR
    A[股價異常跳空] --> B[到 MOPS 查當日重大訊息]
    B --> C{有無重大事件?}
    C -->|有| D[判斷利多/利空與是否已反映]
    C -->|無| E[轉查籌碼/盤面/同業]

    這是只看落後財報數字、或依賴合成資料所無法取代的查證習慣。建構 深入分析 或 觀察清單 時尤其重要。

    言行反查¶

    flowchart LR
    A[法說樂觀] --> B{法人籌碼?}
    B -->|買超| C[較一致]
    B -->|賣超| D[提高警覺 深入查證]
    D --> E[是短線調節還是結構性出貨?]
    現象 可能意義
    口頭看多 + 法人連賣 話術與行為不一致
    口頭保守 + 法人連買 可能低調佈局
    新聞標題聳動 先看原始公告全文

    常見話術¶

    話術 應對
    「審慎樂觀」 看有無量化目標
    「符合預期」 對照市場預期是高是低
    「短期波動」 看是否連續多季下修

    常見誤區¶

    誤區 正確做法
    只看新聞標題 到 MOPS 重大訊息 讀原文
    法說樂觀就追 對照 法人表 言行是否一致
    有公告 = 還沒反映 股價常領先；要判斷利多是否已反映在價格上
    忽略 Q&A 分析師追問的尖銳問題常藏風險

    自我檢查¶

    1.（概念題）法說會閱讀清單的第一項是什麼？

    參考答案：看管理層有無量化指引（營收、毛利率、CapEx 等具體數字）。

    2.（判斷題）口頭看多 + 法人連續賣超，該更放心還是更警覺？

    參考答案：更警覺——話術與行為不一致，需深入查證是否結構性出貨。

    3.（情境題）股價跳空，你第一步該做什麼？

    參考答案：到 MOPS 查當日重大訊息 → 判斷利多／利空與是否已反映；見 事件驅動查證。

    重點回顧¶

    • 法說與公告是文字面，籌碼是行為面，兩者應交叉驗證。
    • 標題新聞不如原始公告可靠。
    • 不構成買賣建議；見 免責聲明。

# 資料來源說明

## 本篇你會學到

- 台股常見公開資料從哪裡來
- 各資料源的更新頻率與限制

## 官方與交易所

| 來源 | 網址 | 常見資料 |
|------|------|----------|
| **公開資訊觀測站（MOPS）** | [mops.twse.com.tw](https://mops.twse.com.tw) | 財報、月營收、重大訊息、法說會 |
| **台灣證券交易所** | [twse.com.tw](https://www.twse.com.tw) | 上市行情、三大法人、融資融券 |
| **櫃買中心** | [tpex.org.tw](https://www.tpex.org.tw) | 上櫃行情與法人 |

## 行情與歷史資料

| 來源 | 說明 |
|------|------|
| TWSE OpenAPI | 日 K、估值、部分籌碼（程式常用） |
| 券商看盤軟體 | 即時報價、五檔（需開戶） |

## 籌碼與持股

| 資料 | 更新 | 注意 |
|------|------|------|
| 三大法人買賣超 | T+1 | 非即時 |
| 融資融券餘額 | T+1 | 口徑依交易所 |
| 集保股權分散表 | 每週 | 適合中線 |

## 基本面

| 資料 | 來源 |
|------|------|
| 月營收 | MOPS，次月 10 日前 |
| 季報 | MOPS，季後法定時限 |
| 除權息 | 交易所公告 |

## 跨市場（參考用）

| 來源 | 資料 |
|------|------|
| yfinance 等 | 美股、指數、部分 ADR |

## 使用原則

1. **以官方公告為準**：媒體標題僅作線索。
2. **注意 lag**：籌碼 T+1、集保週更，不適合當唯一即時依據。
3. **口徑一致**：不同網站 PER、法人合計可能略有差異，比較時用同一來源。

## 重點回顧

- MOPS + 交易所是台股基本面與籌碼的權威來源。
- 即時交易需券商行情；本教學站不要求即時資料。

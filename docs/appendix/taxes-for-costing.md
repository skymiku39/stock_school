# 稅費總覽（供成本試算）

## 本篇你會學到

- 台股投資各環節會遇到哪些**稅與費**
- 試算損益、殖利率、期望值時該扣哪些項目
- 各項延伸到哪一章看細節

[← 基本資料書](reference-book.md) · [交易成本](../06-risk/trading-costs.md) · [公式速查](formulas.md)

!!! warning "定位"
    本頁**不含**綜所稅申報、扣繳憑單填寫等流程；**不含**每月**勞保、全民健保費**（屬薪資／固定支出，算 [閒錢](../06-risk/capital.md#閒錢與生活費) 時先扣）。**股利領息**時的**二代健保**補充保費已列入下方總表。其餘只整理「有哪些稅費、何時影響投資成本與實領金額」。費率與門檻依**當年法規、券商公告**為準，試算前請對照 [交割與稅費](../01-basics/settlement-fees.md) 與券商對帳單。

---

## 一頁總表

| 環節 | 項目 | 英文 | 性質 | 試算時怎麼用 |
|------|------|------|------|--------------|
| **買進** | 手續費 | Brokerage fee | 費 | 成交金額 × 折扣後費率；買進當下扣 |
| **賣出** | 手續費 | Brokerage fee | 費 | 同上；賣出時再扣一次 |
| **賣出（個股）** | 證交稅 | Securities transaction tax | 稅 | 賣出金額 × **0.3%**（僅賣出） |
| **賣出（ETF）** | 證交稅 | Securities transaction tax | 稅 | 賣出金額 × **0.1%** |
| **現股當沖賣出** | 證交稅（優惠） | Day-trade STT | 稅 | 賣出金額 × **0.15%**（須符合資格；優惠期至 **2027 年底**，非永久） |
| **持有 ETF** | 內扣費用 | Expense ratio | 費 | 每日自淨值扣除；長期持有必算，見 [ETF 費用](../01-basics/etf-costs-and-premium.md) |
| **除權息** | 過戶費 | Transfer fee | 費 | 領股利／配股時券商收取；試算自對帳單帶入 |
| **領現金股利** | 股利所得稅 | Dividend income tax | 稅 | 影響**實領現金**；試算勿只用帳面殖利率 |
| **領現金股利** | 二代健保補充保費 | NHI supplementary premium | 費/扣款 | 單次股利達門檻時可能另扣 **2.11%**（現行常見門檻 **2 萬元**；費率依 [健保署](https://www.nhi.gov.tw) 當年度公告） |
| **融資** | 利息 | Margin interest | 成本 | 非稅，但算淨利必加；見 [融資融券](../02-glossary/chips.md#融資融券) |
| **期貨** | 期貨交易稅 | Futures transaction tax | 稅 | 依期交所／法規；見 [期貨入門](../01-basics/futures-intro.md) |

**費** = 券商或機構收取；**稅** = 依法課徵。試算**淨利**時兩者都要扣。

## 依環節說明

### 1. 買賣成交（現股 / ETF）

| 時點 | 扣什麼 | 備註 |
|------|--------|------|
| 買進成交 | 股款 + 手續費 | 買進**不**收證交稅 |
| 賣出成交 | 手續費 + 證交稅 | 證交稅只在**賣出**時課徵 |
| T+2 交割 | 款項交收 | 不另加稅，但影響資金可用日 |

**損益平衡（個股現股，示意）**

```
來回成本% ≈ 買進手續費% + 賣出手續費% + 證交稅%（賣出）
         ≈ 0.04% + 0.04% + 0.3% ≈ 0.38%（28 折手續費假設）
```

**損益平衡（現股當沖，示意）**

```
來回成本% ≈ 0.04% + 0.04% + 0.15% ≈ 0.23%
```

完整費率表與數字範例 → [交易成本與期望值](../06-risk/trading-costs.md#費用結構)。

---

### 2. ETF 持有（非買賣當下）

| 項目 | 試算重點 |
|------|----------|
| 買賣證交稅 | 賣出 **0.1%**，低於個股 0.3% |
| 內扣管理費 | 從淨值每日扣除，**長期定額必算** |
| 折溢價 | 非稅，但影響買入成本 |

三層費用圖 → [ETF 費用與折溢價](../01-basics/etf-costs-and-premium.md)。

---

### 3. 除權息（存股 / 領息）

除權息當下股價會調整，**總資產大致不變（未計稅）**；真正影響報酬的是**入帳後扣掉的費稅**。

| 項目 | 說明 | 試算建議 |
|------|------|----------|
| **過戶費** | 除權息過戶時券商／集保收取 | 自券商對帳單帶入；勿假設為 0 |
| **股利所得稅** | 現金股利屬所得；可採分離課稅或合併綜所稅 | 試算用**實領股利**；分離課稅常以 **28%** 作保守上限參考（合併申報者實際可能不同，本站不展開） |
| **二代健保** | 單次股利達門檻時可能扣 **2.11%** | 大額股利試算須預留；門檻與例外依 [健保署](https://www.nhi.gov.tw) 當年公告 |

**實領殖利率（示意）**

```
實領殖利率 ≈ (現金股利 − 過戶費 − 稅費預估) ÷ 投入成本 × 100%
```

帳面殖利率 `股利 ÷ 股價` **未扣上述項目**，不可直接當「口袋裡的報酬」。

延伸：[除權息入門](../01-basics/dividend.md) · [填息](../02-glossary/market-terms.md#填息) · [除權息案例](../07-cases/dividend-play.md)

---

### 4. 信用交易（融資 / 融券）

| 項目 | 性質 | 試算 |
|------|------|------|
| 融資利息 | 成本（非稅） | 持有天數 × 利率 × 借款額 |
| 融券借券費 | 成本（非稅） | 依券商與標的 |
| 買賣證交稅 | 同現股 | 平倉賣出時仍課證交稅 |

---

### 5. 期貨

| 項目 | 試算 |
|------|------|
| 期貨交易稅 | 依現行期交稅率 × 契約金額（以期交所公告為準） |
| 手續費 | 依期貨商定額或折扣 |

見 [期貨入門](../01-basics/futures-intro.md)。

## 試算公式速查

| 目的 | 公式 | 備註 |
|------|------|------|
| 買進總成本 | `股價 × 1,000 × 張數 + 買進手續費` | [張](../02-glossary/position.md) |
| 賣出淨收入 | `賣出金額 − 賣出手續費 − 證交稅` | 證交稅率依標的與當沖與否 |
| 交易淨利% | `(賣出淨收入 − 買進總成本) ÷ 買進總成本 × 100%` | 即 [淨利](../02-glossary/pnl.md#淨利) 概念 |
| 損益平衡漲幅 | `買進手續費% + 賣出手續費% + 證交稅%` | 當沖用 0.15% 稅率 |
| 帳面殖利率 | `現金股利 ÷ 股價 × 100%` | **未扣**股利稅費 |
| 實領殖利率 | `(股利 − 過戶費 − 稅費預估) ÷ 成本 × 100%` | 存股試算用 |

更多算式 → [公式速查](formulas.md#損益與風控)。

## 常見試算錯誤

| 錯誤 | 正確做法 |
|------|----------|
| 只用漲跌幅算獲利 | 扣掉**來回手續費 + 賣出證交稅** |
| 當沖用 0.3% 證交稅 | 符合資格時用 **0.15%**（注意優惠期限） |
| ETF 只看 0.1% 證交稅 | 長期還要算**內扣費** |
| 殖利率 5% 就以為賺 5% | 用**實領股利**試算，預留過戶費、股利稅、二代健保 |
| 買進當下就算證交稅 | 證交稅只在**賣出**時課徵 |

## 綜所稅與申報邊界 {#綜所稅與申報邊界}

本站**刻意不教**綜合所得稅申報、扣繳憑單填寫、海外所得申報等流程（見 [STYLE-GUIDE 邊界主題](../STYLE-GUIDE.md#邊界主題c-級即達標不追求操作-a)）。本頁只幫你試算**交易與股利領取當下**該扣哪些稅費。

### 何時自行試算就夠

| 情境 | 建議 |
|------|------|
| 現股買賣損益、當沖成本 | 用本站 [交易成本](../06-risk/trading-costs.md) 與上方公式 |
| 領息試算（過戶費、二代健保） | 用**實領殖利率**粗估 |
| 單一薪資 + 單一券商、股利採分離課稅 | 多數人可用券商對帳單 + 當年公告粗算 |

### 何時建議諮詢會計師或國稅局

| 情境 | 原因 |
|------|------|
| 股利要選**合併綜所稅**還是**分離課稅**拿不準 | 兩者對總稅負影響大，因人而異 |
| 多券商、多帳戶、有**海外股票**或複雜所得 | 申報口徑與扣抵易漏 |
| 有大額**股利、營業所得、贈與**等跨年度規劃 | 超出本站試算範圍 |
| 被通知補稅、稽查或法規異動不確定 | 需專業解讀當年法令 |

### 站外官方查詢入口

| 來源 | 用途 |
|------|------|
| [財政部稅務入口網](https://www.etax.nat.gov.tw) | 當年稅法、股利所得、申報期間 |
| [國稅局 e 化服務](https://www.etax.nat.gov.tw/etwmain) | 個人資料、試算工具（依當年開放項目） |
| 券商對帳單／年度憑單 | 股利、證交稅、二代健保實際扣款 |

!!! note "與投資試算的關係"
    即使不自行申報，**試算淨利與實領殖利率**仍應預留股利稅與二代健保；精算年度總稅負則交給專業或官方工具。

## 自我檢查

??? question "1.（概念題）證交稅在買進還是賣出時課徵？"
    參考答案：僅在**賣出**時課徵；買進只扣手續費。

??? question "2.（判斷題）帳面殖利率 5% 就等於口袋裡賺 5%？"
    參考答案：否。須扣過戶費、股利所得稅、可能二代健保，用**實領殖利率**試算。

??? question "3.（情境題）你有海外股票與多張券商單，年度申報該怎麼做？"
    參考答案：超出本站範圍，建議諮詢**會計師**或查 [財政部稅務入口網](https://www.etax.nat.gov.tw)；本站只做買賣與領息當下成本試算。

## 重點回顧

- **交易**：手續費（買+賣）+ 證交稅（僅賣出）；當沖、ETF、個股稅率不同。
- **領息**：過戶費 + 股利所得稅 + 可能二代健保 → 用**實領**試算，勿只看帳面殖利率。
- **不含申報**：本站只幫你列清「該扣什麼」；實際稅負因人、因年度而異，精算請查當年法規或諮詢專業。

# 投資檢查清單

## 本篇你會學到

- 下單前、除權息前、每月複盤各該檢查什麼
- 把分散的紀律濃縮成可勾選的清單

!!! note "怎麼用"
    這是**學員實戰版**檢查清單，列印或截圖隨手對照即可。維護者用的內容審查版見 [投資審查檢核表](../INVESTMENT-REVIEW-CHECKLIST.md)。

## 一、下單前 10 項

- [ ] **交割能力**：交割戶的錢夠 T+2 扣款？（[交割與費用](../01-basics/settlement-fees.md)）
- [ ] **委託方式**：限價還是市價？價格合理？
- [ ] **部位大小**：單檔金額在我的 [曝險上限](../06-risk/capital.md) 內？
- [ ] **停損點**：進場前已寫好停損淨利（如 −3%）？（[停損三層](../06-risk/stop-loss.md)）
- [ ] **停利／減碼**：目標與減碼規則想好了？（[停利與減碼](../06-risk/stop-loss.md#停利與減碼)）
- [ ] **交易標記**：有無 [注意／處置／全額交割](../01-basics/trading-restrictions.md)？
- [ ] **成本估算**：手續費 + 證交稅後的 [損益兩平點](../02-glossary/pnl.md)？
- [ ] **投資論點**：我為什麼買？一句話講得出來？（[研究流程](../09-advanced/research-workflow.md)）
- [ ] **資料佐證**：營收／籌碼／技術至少一面支持？（[怎麼看表](../03-tables/index.md)）
- [ ] **這是閒錢嗎**：不是生活費或借來的錢？（[閒錢與生活費](../06-risk/capital.md#閒錢與生活費)）

## 二、除權息前 10 項

- [ ] 確認 [除權息日與最後過戶日](../03-tables/dividend-schedule.md)。
- [ ] 看清是**除息、除權或兩者**。
- [ ] 估算除權息**參考價**（除息約減現金股利）。
- [ ] 評估**填息機率**：公司獲利、過往填息紀錄。
- [ ] 殖利率是否被股價下跌**被動墊高**？（[估值陷阱](../07-cases/valuation-trap.md)）
- [ ] 是否只為「賺股利」短進短出（股價多已反映）？
- [ ] 考量**股利所得稅**與二代健保補充保費 **2.11%**（單次 ≥2 萬；見 [稅費總覽](taxes-for-costing.md)）。
- [ ] 參與後是否打亂原本的 [資金配置](../06-risk/capital.md)？
- [ ] 是否誤把除息跳空當利空？（[除息誤買](../06-risk/emergency-playbook.md#除息誤買)）
- [ ] 已讀 [除權息入門](../01-basics/dividend.md) 與 [案例](../07-cases/dividend-play.md)？

## 三、每月複盤 10 項

- [ ] 本月**淨損益**（含費稅）是多少？
- [ ] 每筆交易**有沒有照計畫**執行停損停利？
- [ ] 賺賠的主因是**判斷對**還是**運氣**？
- [ ] 最大單筆虧損是否超出預設？檢視 [最大回撤](../02-glossary/risk.md)。
- [ ] 是否出現追高殺低、凹單等**情緒交易**？（[交易紀律](../06-risk/discipline.md)）
- [ ] 持股的**投資論點**是否仍成立？
- [ ] 組合是否需要 [再平衡](../09-advanced/portfolio.md#再平衡)？
- [ ] 曝險與現金比例是否健康？
- [ ] 有沒有可重複的**錯誤模式**要改？（[老手陷阱](../09-advanced/veteran-pitfalls.md)）
- [ ] 把心得寫進**交易日誌**了嗎？

## 自我檢查

??? question "1.（概念題）三份清單分別對應哪三個時點？"
    參考答案：**下單前**（控風險）、**除權息前**（防誤判）、**每月複盤**（求進步）。

??? question "2.（判斷題）清單勾完就可以不用設停損？"
    參考答案：不行。下單前第 4 項要求**進場前寫好停損淨利**並執行。

??? question "3.（情境題）除息當天股價跳空下跌，除權息前清單哪幾項該先回想？"
    參考答案：參考價技術調整、勿誤當利空；見清單第 3、9 項與 [除息誤買](../06-risk/emergency-playbook.md#除息誤買)。

## 重點回顧

- 三個時點、各 10 項：下單前控風險、除息前防誤判、月複盤求進步。
- 清單的價值在於**每次都用**，把紀律變成習慣。

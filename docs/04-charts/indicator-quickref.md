## 時間框架建議

| 框架 | 優先指標 |
|------|----------|
| 當沖 | MA5/10、KD、成交量 |
| 短線 | MA20、MACD、RSI |
| 中線 | MA60、MACD、布林 |
| 長線 | MA120/240、基本面（非指標） |

## 搭配原則

```mermaid
flowchart LR
    A[趨勢: MA/MACD] --> C[決策]
    B[時機: RSI/KD/布林] --> C
    D[量: 成交量] --> C
```

- **趨勢指標**（MA、MACD）告訴你「大方向」
- **震盪指標**（RSI、KD）告訴你「短線冷熱」
- **布林**告訴你「波動是否變大」
- 三者同向時訊號較一致，但仍需 [停損](../06-risk/stop-loss.md)

## 自我檢查

??? question "1.（概念題）MA/MACD 與 RSI/KD 各回答什麼？"
    參考答案：**趨勢**（大方向）vs **震盪**（短線冷熱）；詳見各指標教學頁。

??? question "2.（判斷題）五個指標都偏多，可以不加停損全倉？"
    參考答案：不行。指標**滯後**且可能假訊號；仍須 [停損](../06-risk/stop-loss.md) 與位置判斷。

??? question "3.（情境題）當沖族優先盯哪幾個指標？"
    參考答案：**MA5/10、KD、成交量**；波段則重 MA+MACD，見本篇時間框架表。

## 重點回顧

- 沒有萬用指標；當沖重 MA+量，波段重 MA+MACD。
- 指標是滯後或統計工具，不能取代 [K 線位置](candle-patterns.md) 判斷。

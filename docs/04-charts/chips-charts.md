## 法人累計買賣超圖

| 圖形特徵 | 解讀 |
|----------|------|
| 累計線向上 | 近期法人淨買超為主 |
| 累計線向下 | 近期法人淨賣超為主 |
| 股價漲 + 累計向上 | 籌碼與價格一致 |
| 股價漲 + 累計向下 | 價漲量縮籌碼，警惕 |

## 融資餘額趨勢圖

| 圖形特徵 | 解讀 |
|----------|------|
| 融資餘額攀升 + 股價漲 | 散戶槓桿增加 |
| 融資餘額急降 + 股價跌 | 可能斷頭或停損潮 |
| 融資餘額橫盤 | 籌碼安定 |

## 集保大戶持股變化

週資料，適合**中線**：

- 大戶（400–1000 張以上）持股增加 → 籌碼集中。
- 散戶級距增加 → 籌碼分散。

## 常見誤區

| 誤區 | 正確認知 |
|------|----------|
| 法人買超圖向上就追 | 資料 T+1，且可能是指數調整 |
| 融資增 = 散戶愚蠢 | 有時是趨勢中的槓桿參與 |
| 把籌碼圖當即時 tick | 法人、融資、集保皆有 **lag** |

## 自我檢查

??? question "1.（概念題）股價漲但法人累計線向下，代表什麼？"
    參考答案：**價籌背離**，價漲但法人近期淨賣，宜提高警覺。

??? question "2.（判斷題）融資餘額急降一定是壞事？"
    參考答案：不一定。可能是斷頭潮（偏空），也可能是主動降槓桿；要對照股價與 [融資表](../03-tables/margin.md)。

??? question "3.（情境題）看中線籌碼，集保與法人哪個較不受單日雜訊干擾？"
    參考答案：**集保週資料**與法人**連續多日**趨勢，優於單日極端值。

## 重點回顧

- 籌碼圖是**趨勢化**呈現表格資料，便於看連續性。
- 背離（價與籌碼相反）值得特別筆記。
- 資料有 lag，勿當即時 tick 使用。

## 本篇你會學到

- 融資餘額、融券餘額、券資比
- 多空情緒的限度

## 欄位解讀

| 欄位 | 意義 | 簡化解讀 |
|------|------|----------|
| **融資餘額** | 尚未還清的融資張數 | 散戶槓桿多寡 |
| **融資增減** | 較前一日變化 | 短線追高或斷頭後減少 |
| **融券餘額** | 尚未還券的融券張數 | 看空或避險部位 |
| **券資比** | 融券/融資（口徑依來源） | 結構參考，非絕對多空 |

## 手算一例 {#手算一例}

以示意表 **6789**：融資餘額 8,000 張、融券餘額 2,000 張，則

```
券資比 = 融券餘額 ÷ 融資餘額 × 100% = 2,000 ÷ 8,000 = 25%
```

券資比偏高代表空方部位相對多，可能是 [軋空](../02-glossary/market-terms.md#軋空) 的溫床，但須搭配股價與流通張數判斷。

## 閱讀步驟

1. **看變化率**：融資／融券是增還是減，比絕對值重要。
2. **配股價方向**：融資增 + 股價漲 vs 融資增 + 股價跌，意義相反。
3. **算券資比**：判斷空方部位輕重（見上方手算）。
4. **連信用風控**：散戶槓桿過熱時，留意 [追繳與斷頭](../06-risk/margin-trading.md)。

## 閱讀情境

| 情境 | 可能意義（需搭配股價） |
|------|------------------------|
| 融資大增 + 股價急漲 | 散戶槓桿追高，留意修正 |
| 融資驟減 + 股價大跌 | 可能斷頭賣壓 |
| 融券增加 + 股價橫盤 | 空方佈局或避險 |
| 券資比極高 | 軋空或結構特殊，需個案分析 |

## 借券賣出

部分資料源另列「借券賣出餘額」，與融券相關但統計不同。重點是：**借券賣出增加**可能代表空方工具使用增加，仍須看股價是否破支撐。

## 常見誤區

- 融資多 = 必跌 → 強勢股融資可長期維持高檔。
- 券資比高 = 必軋空 → 需看流通張數與主力意圖。

## 讀完請做

走一遍 [融券與軋空案例](../07-cases/short-squeeze.md)：看融券餘額與借券如何在利多時引發軋空，並對照 [信用交易實務](../06-risk/margin-trading.md) 的維持率風險。

## 重點回顧

- 融資融券是**籌碼結構**，不是買賣訊號。
- 變化率與股價方向一起讀。
- 小市值股融資占比影響更大。

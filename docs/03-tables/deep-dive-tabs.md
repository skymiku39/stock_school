# 個股深入分析：分頁地圖

## 本篇你會學到

- 單檔股票「深入分析」畫面有哪些分頁、各看什麼
- 從 KPI 到分頁的閱讀順序
- 與本站其他章節的對照

!!! note "說明"
    本頁描述常見看盤／研究工具中「單檔深入」的欄位邏輯，並對照相鄰 Stock Bot 儀表板。你不必安裝任何程式也能依此結構閱讀公開資料。

---

## 頂部：先做這三件事

| 步驟 | 內容 |
|------|------|
| 1 | 確認**代號與名稱**、資料快照時間 |
| 2 | 看 **KPI 橫列**：現價、漲跌%、法人累計、借券變動、ETF 共識檔數 |
| 3 | 看 **四時間框架評分**（短／中短／中／長）→ 詳 [評分量表](scoring.md) |

若出現「缺資料」提示，代表該來源尚未更新；不必一次補齊全部，依你當下問題選擇刷新項目即可。

## 建議閱讀順序

=== "第一次看這檔股"

    1. KPI + 四時間框架評分
    2. **基本面**（營收趨勢、估值是否合理）
    3. **技術面**（趨勢與進場區間）
    4. **籌碼面**（法人與融券是否矛盾）

=== "已有持倉"

    1. **目前狀況**（損益、停損距離）
    2. **分析**（法說或新聞是否改變投資論點（thesis））
    3. **季報**（獲利是否偏離預期）

=== "找 ETF 共識標的"

    1. KPI「ETF 共識」檔數
    2. [主動 ETF 分析](../05-analysis/active-etf.md)
    3. 基本面 + 籌碼交叉驗證

## 各分頁細項（欄位意義）

### 分析分頁

| 區塊 | 用途 |
|------|------|
| 情緒 / 信心 | 文字素材的整體偏多偏空（教學用參考） |
| 成長驅動 / 風險點 | 列出 thesis 的正反因素 |
| 言行反查 | 管理層說法與數據是否一致 |
| 原始素材 | MOPS、新聞、網頁 — 建議**至少掃一次來源** |

ETF 無傳統法說會時，此區可能改為新聞／情緒彙整，解讀時勿當成法說逐字稿。

### 基本面分頁

| 常見欄 | 見 |
|--------|-----|
| MoM / YoY | [營收表](revenue.md) |
| PER / PBR / 殖利率 | [估值表](valuation.md) |

### 技術面分頁

## 與「個股總覽表」的差異

| 畫面 | 目的 |
|------|------|
| [個股總覽表](watchlist.md) | 多檔並排、排序、快速篩選 |
| **個股深入分析**（本頁） | 單檔完整 3D 資料與歷史操作 |

典型流程：總覽表挑出候選 → 深入分析驗證細節 → 決定是否列入觀察清單（watchlist）或下單。

---

## 重點回顧

- 分頁是**同一檔股票的不同切面**；評分卡是濃縮，分頁是證據。
- 缺資料時先問「我這次決策最需要哪一面」，再選擇更新來源。
- ETF 與一般個股在基本面、法說欄位上有所不同，避免用錯模板。

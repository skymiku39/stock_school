## 開倉 / 平倉 / 建倉 {#開倉平倉}

| 用語 | 定義 |
|------|------|
| **開倉（建倉）** | 開始建立一筆交易部位，可做多（買進）或做空（融券賣出） |
| **平倉** | 結束該筆部位的反向操作（賣出持有多單、買回平空單） |
| **加碼 / 減碼** | 在已有部位上增加或減少數量 |

**在哪裡看到**：交易軟體「下單」「平倉」按鈕、投資社群「我昨天開倉」。

**常見誤解**：開倉 = 一定要賺。開倉只是開始，風控與停損同樣重要。

相關：[持倉與交易](position.md) · [交易流程](../01-basics/trading-flow.md)

## 抄底 {#抄底}

**定義**：在股價跌至認為的「極低」或恐慌區間時買進，期待日後回升獲利。

**性質**：屬於**逆勢**操作——價格仍在下跌趨勢中就進場。

**常見誤解**：跌很多就便宜。可能只是「從貴變合理」，甚至「從合理變更便宜」。

!!! warning "新手提醒"
    口袋小學堂等影片多建議新手勿輕易抄底；若要做，需有明確支撐、基本面與 [停損](../06-risk/stop-loss.md) 計畫。

## 追高殺低 {#追高殺低}

**定義**：

| 行為 | 說明 |
|------|------|
| **追高** | 股價已大幅上漲、情緒火熱時才買進 |
| **殺低** | 股價急跌、恐慌時在相對低點賣出 |
| **追高殺低** | 高買低賣的組合，是常見虧損原因 |

**心理**：貪婪（怕錯過）與恐懼（怕繼續跌）驅動，違反「低買高賣」原則。

**改善方向**：[分批](#分批) 進出、[紀律](../06-risk/discipline.md)、事先寫好進出場計畫。

相關案例：[突破與假突破](../07-cases/gap-breakout.md)

## 回檔 / 拉回 {#回檔}

**定義**：股價在**上漲趨勢**過程中，出現暫時性下跌。

| 說法 | 補充 |
|------|------|
| **拉回不破均線** | 下跌但未跌破重要均線 → 趨勢可能延續 |
| **拉回很深** | 回檔幅度大，需留意趨勢是否轉弱 |

**常見誤解**：回檔 = 轉空。在強勢股中，健康回檔是常態。

---

## 反彈 {#反彈}

**定義**：股價在**下跌趨勢**過程中，出現暫時性回升（像球落地彈起）。

| 說法 | 補充 |
|------|------|
| **反彈不過均線** | 雖回升，但未突破均線或壓力 → 空頭可能仍占優 |
| **反彈不過前高** | 技術上弱勢訊號之一 |

回檔與反彈都是**波動中的正常現象**，關鍵在趨勢方向與位置。見 [行情與報價](quotes.md)、[均線](../04-charts/ma.md)。

## 套牢 / 解套 {#套牢解套}

| 用語 | 定義 |
|------|------|
| **套牢** | 買進後股價低於成本，不願認賠而繼續持有，資金被占用 |
| **解套** | 股價回升至成本價或以上，可賣出不虧損 |

**在哪裡看到**：投資社群「我套了 20%」、新聞「套牢賣壓」。

**常見誤解**：只要不賣就不算虧。帳面虧損與機會成本都是真實的。

**與打底的關係**：股價在底部區間反覆震盪時，早期買進者常形成套牢賣壓，見 [市場進階術語](market-terms.md#打底)。

## 分批 {#分批}

| 類型 | 做法 | 目的 |
|------|------|------|
| **分批加碼（買）** | 不一次 All in，分數次買進 | 降低買在尖點的風險 |
| **分批停利（賣）** | 漲一段先賣一部分，其餘續抱 | 鎖定部分獲利、減少貪念 |

**小例子**：計畫買 3 張 → 先 1 張試單，回檔再加 1 張，突破確認後加最後 1 張。

相關：[資金配置](../06-risk/capital.md) · [停損停利](../06-risk/stop-loss.md)

## 類股 {#類股}

**定義**：依**產業**分類的股票群體。

| 台股常見類股 | 說明 |
|--------------|------|
| 電子 / 半導體 | 權值與景氣循環影響大 |
| 金融 | 與利率、景氣相關 |
| 航運、鋼鐵 | 常見景氣與原物料題材 |

**用途**：大盤分析時常說「今天電子強、傳產弱」；選股時可比較**同類股**估值。

相關：[跨市場與產業](../05-analysis/cross-market.md) · [基本面框架](../05-analysis/fundamental-framework.md#行業層次)

## 「回測」兩種意思 {#回測}

同一個詞在投資圈有**兩種常見用法**，閱讀時請看上下文：

| 類型 | 定義 | 常見出處 |
|------|------|----------|
| **策略回測** | 把交易策略套用到**歷史資料**，檢驗過去績效 | 量化、程式交易 |
| **技術回測** | 價格**回頭測試**支撐線或壓力線 | K 線、技術分析 |

| 技術用語 | 意義 |
|----------|------|
| **回測不破支撐** | 跌到支撐附近後反彈，支撐有效 |
| **跌破支撐** | 支撐失效，該價位可能轉為壓力 |

詳見 [技術面術語](technical.md#回測支撐) · [策略回測](../appendix/video-resources.md#口袋小學堂-ep53)

## 重點回顧

- **開倉／平倉**是部位生命週期的起迄；**追高殺低**是應避免的心態組合。
- **回檔**發生在漲勢中，**反彈**發生在跌勢中，皆需看趨勢與均線。
- **分批**可控制貪恐，**類股**幫助你比較同產業標的。
- **回測**一字兩義：策略驗證 vs 價格測試支撐。

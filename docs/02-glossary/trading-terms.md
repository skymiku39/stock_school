## 開倉 / 平倉 / 建倉（Open / Close Position） {#開倉平倉}

| 項目 | 說明 |
|------|------|
| **英文** | Open position (建倉) / Close position |
| **定義** | 開倉（建倉）：開始建立一筆交易部位，可做多（買進）或做空（融券賣出）；平倉：結束該筆部位的反向操作（賣出多單、買回平空單）；加碼／減碼：在已有部位上增加或減少數量 |
| **在哪裡看到** | 交易軟體「下單」「平倉」按鈕、投資社群「我昨天開倉」 |
| **常見誤解** | 開倉 = 一定要賺；開倉只是開始，風控與停損同樣重要 |
| **小例子** | 先開倉買 1 張，回檔加碼 1 張，達目標分批平倉 |

相關：[持倉與交易](position.md) · [交易流程](../01-basics/trading-flow.md)

## 抄底（Bottom Fishing） {#抄底}

| 項目 | 說明 |
|------|------|
| **英文** | Bottom fishing |
| **定義** | 在股價跌至認為的「極低」或恐慌區間時買進，期待日後回升獲利；屬於**逆勢**操作 |
| **在哪裡看到** | 急跌後的社群討論、新聞「低接買盤」 |
| **常見誤解** | 跌很多就便宜；可能只是「從貴變合理」，甚至「從合理變更便宜」 |
| **小例子** | 股價腰斬後進場，但下跌趨勢未止 → 越接越深 |

!!! warning "新手提醒"
    口袋小學堂等影片多建議新手勿輕易抄底；若要做，需有明確支撐、基本面與 [停損](../06-risk/stop-loss.md) 計畫。

## 追高殺低（Chasing Highs, Selling Lows） {#追高殺低}

| 項目 | 說明 |
|------|------|
| **英文** | Chasing highs, selling lows |
| **定義** | 追高：股價已大幅上漲、情緒火熱時才買進；殺低：股價急跌、恐慌時在相對低點賣出；合稱高買低賣，是常見虧損原因 |
| **在哪裡看到** | 情緒交易後的對帳單、社群「又買在高點」 |
| **常見誤解** | 跟著熱度走就安全；貪婪與恐懼正是追高殺低的根源 |
| **小例子** | 利多噴出追在漲停 → 隔日回檔恐慌殺在低點 |

**改善方向**：[分批](#分批) 進出、[紀律](../06-risk/discipline.md)、事先寫好進出場計畫。相關案例：[突破與假突破](../07-cases/gap-breakout.md)。

## 回檔 / 拉回（Pullback） {#回檔}

| 項目 | 說明 |
|------|------|
| **英文** | Pullback |
| **定義** | 股價在**上漲趨勢**過程中，出現暫時性下跌 |
| **在哪裡看到** | 上升趨勢中的短期黑K、跌向均線 |
| **常見誤解** | 回檔 = 轉空；在強勢股中，健康回檔是常態 |
| **小例子** | 拉回不破 MA20 → 趨勢可能延續；拉回很深 → 留意趨勢轉弱 |

---

## 反彈（Rebound） {#反彈}

| 項目 | 說明 |
|------|------|
| **英文** | Rebound |
| **定義** | 股價在**下跌趨勢**過程中，出現暫時性回升（像球落地彈起） |
| **在哪裡看到** | 下跌趨勢中的短期紅K、回升至均線附近 |
| **常見誤解** | 反彈 = 落底回升；反彈不過均線或前高常是弱勢訊號 |
| **小例子** | 反彈不過 MA20 又下跌 → 空頭可能仍占優 |

回檔與反彈都是**波動中的正常現象**，關鍵在趨勢方向與位置。見 [行情與報價](quotes.md)、[均線](../04-charts/ma.md)。

## 套牢 / 解套（Locked-in / Break-even Recovery） {#套牢解套}

| 項目 | 說明 |
|------|------|
| **英文** | Locked-in (underwater) / Break-even recovery |
| **定義** | 套牢：買進後股價低於成本，不願認賠而繼續持有，資金被占用；解套：股價回升至成本價或以上，可賣出不虧損 |
| **在哪裡看到** | 投資社群「我套了 20%」、新聞「套牢賣壓」 |
| **常見誤解** | 只要不賣就不算虧；帳面虧損與機會成本都是真實的 |
| **小例子** | 100 買進跌到 80（套牢），回到 100 才解套 |

**與打底的關係**：股價在底部區間反覆震盪時，早期買進者常形成套牢賣壓，見 [市場進階術語](market-terms.md#打底)。

## 分批（Scaling） {#分批}

| 項目 | 說明 |
|------|------|
| **英文** | Scaling (in tranches) |
| **定義** | 分批加碼（買）：不一次 All in，分數次買進以降低買在尖點的風險；分批停利（賣）：漲一段先賣一部分，其餘續抱以鎖定部分獲利 |
| **在哪裡看到** | 交易計畫、資金配置規則 |
| **常見誤解** | 分批等於攤平；攤平是虧損加碼，分批是事先規劃的進出節奏 |
| **小例子** | 計畫買 3 張 → 先 1 張試單，回檔再加 1 張，突破確認後加最後 1 張 |

相關：[資金配置](../06-risk/capital.md) · [停損停利](../06-risk/stop-loss.md)

## 類股（Sector） {#類股}

| 項目 | 說明 |
|------|------|
| **英文** | Sector |
| **定義** | 依**產業**分類的股票群體（電子／半導體、金融、航運、鋼鐵等） |
| **在哪裡看到** | 產業漲跌幅排行、「今天電子強、傳產弱」 |
| **常見誤解** | 同類股齊漲齊跌；龍頭與二三線分化常見 |
| **小例子** | 比較同類股 PER → 判斷個股相對貴或便宜 |

相關：[跨市場與產業](../05-analysis/cross-market.md) · [基本面框架](../05-analysis/fundamental-framework.md#行業層次)

## 「回測」兩種意思（Backtest / Retest） {#回測}

| 項目 | 說明 |
|------|------|
| **英文** | Backtest（策略） / Retest（支撐壓力） |
| **定義** | 策略回測：把交易策略套用到**歷史資料**檢驗過去績效（量化、程式交易）；技術回測：價格**回頭測試**支撐線或壓力線（K 線、技術分析） |
| **在哪裡看到** | 量化報告、技術分析畫線 |
| **常見誤解** | 兩種「回測」混用；閱讀時請依上下文判斷 |
| **小例子** | 回測不破支撐 → 支撐有效；跌破支撐 → 該價位常轉為壓力 |

詳見 [技術面術語](technical.md#回測支撐) · [策略回測](../appendix/video-resources.md#口袋小學堂-ep53)

## 自我檢查

??? question "1.（概念題）回檔與反彈各發生在什麼趨勢中？"
    參考答案：**回檔**在漲勢中、**反彈**在跌勢中；皆需看趨勢與均線。

??? question "2.（判斷題）「回測」只有一種意思？"
    參考答案：否。有**策略回測**（歷史績效）與**價格回測支撐**兩義，見本篇小節。

??? question "3.（情境題）開倉與平倉各代表什麼？"
    參考答案：開倉＝建立部位；平倉＝反向結束部位；見 [開倉／平倉](trading-terms.md#開倉平倉)。

## 重點回顧

- **開倉／平倉**是部位生命週期的起迄；**追高殺低**是應避免的心態組合。
- **回檔**發生在漲勢中，**反彈**發生在跌勢中，皆需看趨勢與均線。
- **分批**可控制貪恐，**類股**幫助你比較同產業標的。
- **回測**一字兩義：策略驗證 vs 價格測試支撐。

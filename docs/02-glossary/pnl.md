# 損益與停損

## 本篇你會學到

- 毛價差與淨利的差異
- 停損、停利、移動停利的概念
- 三種「建議數字」不要混在一起

---

## 毛價差（Gross Spread）

| 項目 | 說明 |
|------|------|
| **英文** | Gross spread |
| **定義** | 僅用股價計算 `(現價 − 成本均價) / 成本均價`，**未扣**手續費與證交稅 |
| **在哪裡看到** | 部分看盤軟體的「未實現損益%」若未含費用，接近毛價差 |
| **常見誤解** | 帳面賺 2% 就達停利；扣稅費後可能只剩 1% 或更少 |
| **小例子** | 成本 100、現價 102 → 毛價差 +2%；扣費約 0.6% 後淨利約 +1.4% |

---

## 淨利（Net P&L） {#淨利}

| 項目 | 說明 |
|------|------|
| **英文** | Net P&L (profit and loss) |
| **定義** | 扣除手續費、證交稅（當沖稅率另計）後的**實際損益百分比或金額** |
| **在哪裡看到** | 券商對帳單、嚴謹的交易日誌 |
| **常見誤解** | 只看股價漲跌；短線與當沖必須用淨利評估策略 |
| **小例子** | 當沖 2 張、每股賺 1 元，若稅費吃掉 0.5%，小波動可能變白忙一場 |

---

## 停損（Stop Loss） {#停損}

| 項目 | 說明 |
|------|------|
| **英文** | Stop loss |
| **定義** | 預先設定「虧損到某程度就出場」，限制單筆最大虧損 |
| **在哪裡看到** | 交易計畫、風控規則（常見例如 -2%～-3% **淨利**） |
| **常見誤解** | 「再等等會彈回來」；沒有停損，小虧可能變大虧 |
| **小例子** | 500 元 × 2 張，停損 -3% 淨利 → 單筆最大虧損約 3 萬元（見 [資金配置](../06-risk/capital.md)） |

---

## 停利（Take Profit）

| 項目 | 說明 |
|------|------|
| **英文** | Take profit |
| **定義** | 獲利達目標價位或獲利% 時賣出鎖定利潤 |
| **在哪裡看到** | 交易計畫、短線策略的目標價 |
| **常見誤解** | 停利點設太低，贏小虧大；需與停損搭配成合理賠率 |
| **小例子** | 進場 100、目標 +6% → 約 106 分批停利 |

---

## 移動停利（Trailing Stop） {#移動停利}

| 項目 | 說明 |
|------|------|
| **英文** | Trailing stop |
| **定義** | 獲利達一定門檻後，追蹤**淨利高點**；若從高點回撤超過設定幅度（如 2%）則出場 |
| **在哪裡看到** | 當沖與短線策略說明 |
| **常見誤解** | 「啟動門檻」是**開始追蹤**的條件，不是「一到就賣」的固定賣點 |
| **小例子** | 淨利先到 +6% 啟動追蹤，最高到 +8%，回撤 2% 即在約 +6% 出場 |

詳見 [停損三層概念](../06-risk/stop-loss.md)。

---

## 損益兩平點（Break-even Point）

| 項目 | 說明 |
|------|------|
| **英文** | Break-even point |
| **定義** | 扣除買賣成本後，股價至少要漲多少才不算白做 |
| **在哪裡看到** | 短線成本試算、交易日誌 |
| **常見誤解** | 忽略來回手續費與賣出稅，高估短線獲利空間 |
| **小例子** | 來回費稅約 0.6% → 至少要漲 0.6% 才損益兩平 |

---

## 自我檢查

??? question "1.（概念題）毛價差與淨利差在哪？"
    參考答案：毛價差只看股價；**淨利**扣掉手續費與證交稅後才是實際損益。

??? question "2.（手算題）成本 100、現價 102，毛價差 +2%；若費稅約 0.6%，淨利約多少？"
    參考答案：約 **+1.4%**（示意）。

??? question "3.（情境題）短線評估為何必用淨利？"
    參考答案：獲利目標常僅 1%～2%，費稅占比高；見 [交易成本](../06-risk/trading-costs.md)。

## 重點回顧

- 評估策略用**淨利**，不要用毛價差自我安慰。
- 停損保護本金；移動停利讓利潤奔跑但鎖住部分獲利。
- 分析建議、戰情建議、實際交易規則可能是三組不同數字，勿混用。

# 術語詞典

## 本篇你會學到

- 如何查閱本站術語
- 字典模式 vs 分類詳解
- 建議閱讀順序

本詞典將台股投資常見名詞整理為**學員版**：分類頁每個詞條為固定五列表格「英文 → 定義 → 在哪裡看到 → 常見誤解 → 小例子」，[完整詞條總表](dictionary.md) 則附**英文欄**方便中英對照與站內搜尋。

!!! tip "當字典用"
    快速查詞 → **[完整詞條總表](dictionary.md)**（100+ 條，含英文、一句話 + 連結；可搜中、英文與縮寫）  
    算公式 / 查縮寫 → [公式速查](../appendix/formulas.md)、[縮寫對照](../appendix/abbreviations.md)  
    使用說明 → [基本資料書指南](../appendix/reference-book.md)

## 建議閱讀順序

1. [行情與報價](quotes.md) — 看盤軟體第一排數字
2. [持倉與交易](position.md) — 理解你「擁有什麼」
3. [損益與停損](pnl.md) — 算清楚賺賠
4. 依興趣選 [基本面](fundamentals.md) 或 [技術面](technical.md)
5. [法人與籌碼](chips.md) + [風控](risk.md)

| 術語 | 詳見 |
|------|------|
| 開高低收 · 漲跌幅 · 成交量 · 漲跌停 | [行情與報價](quotes.md) |
| 淨利 · 停損 · 移動停利 | [損益與停損](pnl.md) |
| 持倉 · 庫存 · 當沖 | [持倉與交易](position.md) |
| 三大法人 · 融資融券 · 期現價差 | [法人與籌碼](chips.md) |
| 月營收 · EPS · PER | [基本面](fundamentals.md) |
| 均線 MA · MACD · RSI | [技術面](technical.md) |
| 曝險 | [風控](risk.md) |
| 跳空 · 軋空 · 填息 | [市場進階](market-terms.md) |

## 重點回顧

- 術語是溝通工具，重點是**定義一致**，不是背越多越好。
- 遇到不確定的詞，先查本分類，再對照 [怎麼看表](../03-tables/watchlist.md) 或 [怎麼看圖](../04-charts/kline-basics.md)。

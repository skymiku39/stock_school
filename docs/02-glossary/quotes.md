# 行情與報價

## 本篇你會學到

- 開高低收、成交量、漲跌幅的意義
- 委買委賣與漲跌停

---

## 開高低收 {#開高低收}

| 項目 | 定義 |
|------|------|
| **開盤** | 該時段第一筆成交價 |
| **最高** | 該時段最高成交價 |
| **最低** | 該時段最低成交價 |
| **收盤** | 該時段最後一筆成交價 |

**在哪裡看到**：K 線圖、個股報價頁、日行情表。2330／0050 範例見 **[報價畫面](../01-basics/quote-screen.md)**。

**常見誤解**：紅K = 漲、黑K = 跌（台股慣例：紅漲黑跌）。但「紅K」只代表收盤 ≥ 開盤，不一定是相對昨收上漲。

**小例子**：開 100、高 105、低 99、收 104 → 紅K，實體從 100 到 104。

---

## 漲跌幅 {#漲跌幅}

**定義**：`(現價或收盤價 - 昨收) / 昨收 × 100%`

**在哪裡看到**：報價列表第一欄、新聞標題「漲 X%」。

**常見誤解**：漲 5% 不代表「還會再漲 5%」；只是相對昨收的變化。

**小例子**：昨收 200，今收 210 → 漲跌幅 +5%。

---

## 成交量 {#成交量}

**定義**：該時段成交的股數（常見以張或千股顯示）。

**在哪裡看到**：K 線下方柱狀圖、個股總覽表的「量」欄。

**常見誤解**：量大一定好。量大可能是換手、出貨或主力對倒，需搭配價格方向。

**小例子**：突破月線時放量 → 較有說服力；跌停放量 → 可能是恐慌殺出。

---

## 委買 / 委賣

**定義**：尚未成交的掛單價格與張數；委買為買方掛單，委賣為賣方掛單。

**在哪裡看到**：五檔或十檔報價。圖解見 **[報價畫面](../01-basics/quote-screen.md#五檔)**。

**常見誤解**：委買很多就會漲。大單可能撤單，且成交才是結果。

---

## 漲跌停 {#漲跌停}

**定義**：個股當日價格相對昨收的上下限（常見 ±10%）。

**在哪裡看到**：報價顯示「漲停」「跌停」、一字線 K 棒。

**常見誤解**：漲停要追。追漲停風險高，且次日不一定續漲。

---

## 昨收

**定義**：前一交易日最後成交價，為當日漲跌幅計算基準。

**在哪裡看到**：報價頁「參考價」「昨收」。

---

## 平盤與開高、開低 {#開高開低}

| 用語 | 定義 |
|------|------|
| **平盤** | 口語常指昨收價，亦作當日競價與漲跌幅的**基準** |
| **開高** | 開盤價 > 昨收（平盤） |
| **開低** | 開盤價 < 昨收 |

| 組合說法 | 意義 |
|----------|------|
| **開高走低** | 開盤偏高，但盤中下跌 |
| **開低走高** | 開盤偏低，但盤中上漲 |

**常見誤解**：開高一定會漲到底。開高後仍可能一路走低收黑K。

參考影片：[學吧十大名詞](../appendix/video-resources.md#學吧-learningpa-十大名詞)

---

## 前高、前低 {#前高前低}

| 用語 | 定義 |
|------|------|
| **前高** | 目前價位之前最近的一個**相對高點** |
| **前低** | 目前價位之前最近的一個**相對低點** |

**在哪裡看到**：日 K、分 K（江波圖）、畫線支撐壓力。

**用途**：突破前高常視為多方延續；跌破前低則偏空。需搭配成交量。

相關：[支撐與壓力](technical.md#支撐--壓力) · [回測支撐](technical.md#回測支撐)

---

## 重點回顧

- 開高低收描述一根 K 棒的價格範圍；漲跌幅是相對昨收。
- 成交量要與價格方向一起看。
- 漲跌停是當日限制，不是趨勢預測。

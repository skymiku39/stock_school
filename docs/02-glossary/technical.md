# 技術面術語

## 本篇你會學到

- 均線、MACD、RSI、KD、布林通道的定義
- 各指標適合回答什麼問題

K 線是技術面的基礎，建議先讀 [K 線基礎](../04-charts/kline-basics.md)、[三招讀懂 K 線](../04-charts/kline-reading.md)、[16 種型態](../04-charts/candle-patterns.md)。

## 均線（MA） {#均線-ma}

| 項目 | 說明 |
|------|------|
| **英文** | Moving Average (MA) |
| **定義** | 過去 N 日收盤價的平均值，如 MA5、MA20、MA60 |
| **在哪裡看到** | K 線圖上的曲線 |
| **常見誤解** | 均線是未來價格；它是**滯後**的趨勢描述 |
| **小例子** | 收盤站上 MA20 → 短期趨勢偏多（但不保證續漲） |

詳見 [均線教學](../04-charts/ma.md)。

---

## MACD {#macd}

| 項目 | 說明 |
|------|------|
| **英文** | MACD (Moving Average Convergence Divergence) |
| **定義** | 由快線、慢線指數移動平均及其差離（DIF）與訊號線（Signal）組成的趨勢動能指標 |
| **在哪裡看到** | K 線圖下方副圖 |
| **常見誤解** | 金叉必買；盤整時假訊號多 |
| **小例子** | DIF 上穿訊號線（黃金交叉）→ 動能轉強參考 |

詳見 [MACD 教學](../04-charts/macd.md)。

---

## RSI {#rsi}

| 項目 | 說明 |
|------|------|
| **英文** | RSI (Relative Strength Index) |
| **定義** | 相對強弱指標，0–100，常見 14 日；>70 視為超買區、<30 視為超賣區 |
| **在哪裡看到** | 副圖 RSI 曲線 |
| **常見誤解** | 超買一定跌；強勢股可長期停留在高位 |
| **小例子** | RSI 由 30 以下回升 → 短線轉強參考 |

詳見 [RSI 教學](../04-charts/rsi.md)。

---

## KD {#kd}

| 項目 | 說明 |
|------|------|
| **英文** | KD (Stochastic Oscillator) |
| **定義** | 隨機指標，比較收盤價在近期高低區間的位置；K 線與 D 線交叉為常見訊號 |
| **在哪裡看到** | 副圖 KD |
| **常見誤解** | K 在 80 以上只能賣；需搭配趨勢與成交量 |
| **小例子** | 低檔 K 上穿 D（黃金交叉）→ 短線轉強參考 |

詳見 [KD 教學](../04-charts/kd.md)。

---

## 布林通道（Bollinger Bands） {#布林通道}

| 項目 | 說明 |
|------|------|
| **英文** | Bollinger Bands |
| **定義** | 以移動平均為中軌，上下軌為 ±N 倍標準差，股價常在通道內波動 |
| **在哪裡看到** | K 線圖包覆通道 |
| **常見誤解** | 觸下軌必買；趨勢下跌時可能沿下軌行走 |
| **小例子** | 開口收窄後突然放大 → 波動率上升的參考 |

詳見 [布林通道教學](../04-charts/bollinger.md)。

## 超買 / 超賣（Overbought / Oversold） {#超買超賣}

| 項目 | 說明 |
|------|------|
| **英文** | Overbought / Oversold |
| **定義** | 市場因消息或情緒，出現**過度買進**（超買）或**過度賣出**（超賣）的狀態 |
| **在哪裡看到** | RSI、KD 進入高位或低位區間 |
| **常見誤解** | 超買一定跌；強勢股可長期超買，需搭配趨勢與 [回檔](trading-terms.md#回檔) 判斷 |
| **小例子** | 利空恐慌後 RSI < 30 → 超賣，留意反彈但非保證落底 |

| 面向 | 說明 |
|------|------|
| **情緒面** | 利多狂歡、利空恐慌，常見於新聞事件後 |
| **指標面** | RSI、KD 進入高位或低位區間 |

參考：[RSI](#rsi) · [口袋 EP53](../appendix/video-resources.md#口袋小學堂-ep53)

## 支撐 / 壓力（Support / Resistance） {#支撐--壓力}

| 項目 | 說明 |
|------|------|
| **英文** | Support / Resistance |
| **定義** | 歷史上股價多次反彈的價位（支撐）或受阻回落的價位（壓力） |
| **在哪裡看到** | K 線畫線、前高前低、整數關卡 |
| **常見誤解** | 支撐永不破；放量跌破支撐可能轉為壓力（支撐變壓力，像天花板） |
| **小例子** | 多次在 100 附近止跌 → 100 為支撐；跌破後 100 常變壓力 |

---

## 回測（支撐）（Retest） {#回測支撐}

| 項目 | 說明 |
|------|------|
| **英文** | Retest (support / resistance) |
| **定義** | 價格回落或反彈後，**回頭測試**某一條支撐線或壓力線 |
| **在哪裡看到** | 突破後拉回測試頸線、均線回測 |
| **常見誤解** | 與策略回測混淆；此處指價格測試支撐壓力 |
| **小例子** | 回測不破支撐 → 支撐有效；跌破支撐 → 該價位常轉為壓力 |

!!! note "一字兩義"
    「回測」亦指把交易策略套用到歷史資料，見 [交易行為術語](trading-terms.md#回測)。

參考：[學吧十大名詞](../appendix/video-resources.md#學吧-learningpa-十大名詞)

## 頭肩頂（Head and Shoulders） {#頭肩頂}

| 項目 | 說明 |
|------|------|
| **英文** | Head and shoulders |
| **定義** | 常見的**反轉型態**：走勢形成左肩、頭部（最高點）、右肩，底部連線為**頸線** |
| **在哪裡看到** | 日線、週線高檔的三峰結構 |
| **常見誤解** | 形狀像就是頭肩頂；盤整中類似形狀可能只是噪音 |
| **小例子** | 頸線放量跌破 → 常被視為趨勢轉空訊號（需搭配量） |

相關：[K 線組合型態](../04-charts/candle-combinations.md) · [16 種型態](../04-charts/candle-patterns.md)

## 型態（Chart Pattern） {#型態}

| 項目 | 說明 |
|------|------|
| **英文** | Chart pattern |
| **定義** | 價格走勢在圖上形成的**形狀**（如頭肩頂、雙底、三角形） |
| **在哪裡看到** | K 線整體結構、趨勢線之間 |
| **常見誤解** | 型態一定兌現；型態突破才是關鍵，且需量能配合 |
| **小例子** | 型態不破 → 趨勢可能延續；型態突破 → 可能開啟新方向 |

**學員建議**：先熟練少數實用型態（單根 [16 種](../04-charts/candle-patterns.md)、[組合](../04-charts/candle-combinations.md)），再擴充。

## 黃金交叉 / 死亡交叉（Golden / Death Cross）

| 項目 | 說明 |
|------|------|
| **英文** | Golden cross / Death cross |
| **定義** | 黃金交叉：短期均線向上穿越長期均線；死亡交叉：短期均線向下穿越長期均線 |
| **在哪裡看到** | 均線組合、MACD 快慢線 |
| **常見誤解** | 交叉就是買賣訊號；盤整時假訊號多，需搭配趨勢 |
| **小例子** | MA5 上穿 MA20 → 黃金交叉，短線轉強參考 |

## 自我檢查

??? question "1.（概念題）技術指標為何要標示滯後性？"
    參考答案：指標由**過去價格**計算，反應慢於當下；強勢股可能超買續漲。

??? question "2.（判斷題）出現黃金交叉就應立即買進？"
    參考答案：否。盤整時假訊號多，需搭配**趨勢與位置**（高檔／低檔）。

??? question "3.（情境題）RSI 與 MACD 各偏回答什麼？"
    參考答案：RSI 偏**震盪冷熱**；MACD 偏**趨勢動能**；宜互相驗證，見 [指標速查](../04-charts/indicator-quickref.md)。

## 重點回顧

- 技術面描述**價格與量能的行為**，不直接告訴你公司好不好。
- 指標互相驗證優於單一訊號。
- 型態與指標需搭配**位置**（高檔 / 低檔 / 趨勢中）。

# 總經與利率術語

## 本篇你會學到

- 國債、升息、降息等總經名詞的白話意義
- 為什麼利率與資金環境會影響股市估值
- 這些詞在新聞與大盤判讀裡怎麼用

!!! note "前置與定位"
    本頁是**總經入門詞庫**，聚焦「定義 + 為何影響股市」。實戰應用見 [總經與類股輪動](../09-advanced/macro-rotation.md) 與 [基本面框架的宏觀層](../05-analysis/fundamental-framework.md#宏觀層次)。

---

## 國債（Government Bond） {#國債}

| 項目 | 說明 |
|------|------|
| **英文** | Government bond / Treasury |
| **定義** | 政府為籌資而發行的債券，到期還本付息；美國公債（US Treasury）的殖利率是全球資金的「無風險報酬」參考點 |
| **在哪裡看到** | 財經新聞「美國 10 年期公債殖利率」、總經數據頁 |
| **常見誤解** | 國債與股票無關；實際上**公債殖利率上升**時，資金可能從股市流向相對安全的債券，壓抑股票估值 |
| **小例子** | 10 年期公債殖利率從 3% 升到 5% → 無風險報酬變高 → 高本益比成長股相對吸引力下降 |

---

## 升息（Rate Hike） {#升息}

| 項目 | 說明 |
|------|------|
| **英文** | Rate hike (interest rate increase) |
| **定義** | 央行（如美國 Fed、台灣央行）調高政策利率，使借錢成本上升、市場資金趨緊 |
| **在哪裡看到** | 央行利率決議新聞、[資金行情](market-terms.md#資金行情) 討論 |
| **常見誤解** | 升息股市一定跌；市場常**提前反映預期**，實際公布時反而未必照走 |
| **小例子** | 連續升息 → 企業利息負擔加重、估值承壓，資金常偏向防禦與高股息 |

升息環境的類股取捨見 [總經與類股輪動](../09-advanced/macro-rotation.md)。

---

## 降息（Rate Cut） {#降息}

| 項目 | 說明 |
|------|------|
| **英文** | Rate cut (interest rate decrease) |
| **定義** | 央行調降政策利率，降低借貸成本、釋放資金，常為刺激景氣 |
| **在哪裡看到** | 央行決議、[資金行情](market-terms.md#資金行情) |
| **常見誤解** | 降息必利多；若降息是因景氣衰退，獲利下修可能抵銷資金面利多 |
| **小例子** | 降息循環初期 + 景氣未壞 → 常見「資金行情」推升大盤 |

---

## 槓桿（Leverage） {#槓桿}

| 項目 | 說明 |
|------|------|
| **英文** | Leverage |
| **定義** | 用借來的資金或保證金放大部位，**同時放大獲利與虧損**的比例 |
| **在哪裡看到** | 融資（[信用交易](../06-risk/margin-trading.md)）、期貨保證金（[期貨入門](../01-basics/futures-intro.md)）、公司財報的 [負債比](../03-tables/financials.md#資產負債表) |
| **常見誤解** | 槓桿只看放大獲利；下跌時同樣放大虧損，且可能 [斷頭](../06-risk/margin-trading.md) |
| **小例子** | 自備 40 萬 + 融資 60 萬買 100 萬股票 → 約 2.5 倍槓桿；跌 20% 自有資金虧 50% |

槓桿出現在三個層面：個人（融資）、衍生品（期貨）、公司（負債）。三者風險邏輯相同——**好的時候更好，壞的時候更壞**。

---

## 對衝 / 避險（Hedge） {#對衝避險}

| 項目 | 說明 |
|------|------|
| **英文** | Hedge |
| **定義** | 用一個方向相反的部位，抵銷既有部位的部分風險，犧牲部分報酬換取穩定 |
| **在哪裡看到** | 法人「自營商避險」欄（[三大法人表](../03-tables/institutional.md)）、期貨避險 |
| **常見誤解** | 對衝 = 看空；對衝是**降低波動**，不一定是看壞後市 |
| **小例子** | 持有一籃子現股，同時放空台指期 → 大盤大跌時期貨獲利抵銷部分現股虧損 |

!!! note "本站範圍"
    散戶對衝多用期貨，屬進階主題。本站以**現股**為主，對衝只作概念說明，操作見 [期貨輔助訊號](../09-advanced/futures-signal.md)。

## 強制平倉與清算（Liquidation） {#強制平倉與清算}

英文 `liquidation` 在投資語境有**兩種**常見意思，勿混淆：

| 意思 | 英文 | 定義 | 在哪裡看到 |
|------|------|------|------------|
| **強制平倉 / 斷頭** | Forced liquidation | 信用或保證金部位維持率不足，被券商/期貨商**強制賣出**還款 | [信用交易實務](../06-risk/margin-trading.md)、[期貨入門](../01-basics/futures-intro.md) |
| **公司清算** | Company liquidation | 公司結束營運、變賣資產償債，股東通常**最後且常拿不回本金** | 下市、重整、破產新聞 |

| 項目 | 說明 |
|------|------|
| **常見誤解** | 看到 liquidation 都當斷頭；公司清算是**企業層級**的終局，與個人部位斷頭不同 |
| **小例子** | 「融資戶遭 liquidation」= 斷頭；「公司進入 liquidation」= 清算下市 |

---

## 選擇權（Options） {#選擇權}

| 項目 | 說明 |
|------|------|
| **英文** | Options (call / put) |
| **定義** | 一種衍生性商品合約，買方付權利金，取得在特定價格買進（call）或賣出（put）標的的**權利**（非義務） |
| **在哪裡看到** | 期權報價、台指選擇權 |
| **常見誤解** | 選擇權像股票一樣單純；其損益受時間價值、波動率影響，**賣方風險可能無限** |
| **小例子** | 買進 call 看漲：付權利金，漲超過履約價才獲利，最大虧損為權利金 |

!!! warning "本站不教選擇權操作"
    選擇權涉及希臘字母、波動率等進階概念，**超出本站現股教學範圍**。本條僅供你看懂新聞名詞；若要實際操作，請另找專門教材並充分理解風險。

---

## 重點回顧

- 國債殖利率、升息、降息決定**資金環境**，是大盤估值的背景。
- 槓桿、對衝、選擇權是**放大或調節風險**的工具，邏輯是「拿穩定或報酬交換」。
- `liquidation` 要分清是個人**斷頭**還是公司**清算**。

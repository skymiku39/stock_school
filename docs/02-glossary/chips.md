## 三大法人（Three Institutional Investors） {#三大法人}

| 項目 | 說明 |
|------|------|
| **英文** | Three institutional investors |
| **定義** | **外資**、**投信**、**自營商**三類機構的買賣超金額或股數 |
| **在哪裡看到** | 公開資訊觀測站、看盤軟體「法人買賣超」表 |
| **常見誤解** | 單日買超就買進；應看**連續多日**與股價是否配合 |
| **小例子** | 連續 5 日外資買超 + 股價站上月線 → 籌碼與價格較一致 |

詳見 [三大法人表](../03-tables/institutional.md)。

## 融資融券（Margin Trading & Short Selling） {#融資融券}

融資與融券是兩種**信用交易**：一個借錢買、一個借股賣。

### 融資（Margin Buying） {#融資}

| 項目 | 說明 |
|------|------|
| **英文** | Margin buying |
| **定義** | 向券商**借錢**買股票，須付利息與維持率 |
| **在哪裡看到** | 融資餘額表、個股融資使用率 |
| **常見誤解** | 融資多 = 看多；也可能是散戶槓桿過高，後續有斷頭賣壓 |
| **小例子** | 融資餘額急增 + 股價滯漲 → 留意追繳與斷頭風險 |

### 融券（Margin Short Selling）

| 項目 | 說明 |
|------|------|
| **英文** | Margin short selling |
| **定義** | 向券商**借股票**賣出，預期價跌後買回還券 |
| **在哪裡看到** | 融券餘額、券資比 |
| **常見誤解** | 融券增加一定會跌；有時是避險或套利，需搭配股價解讀 |
| **小例子** | 融券餘額高 + 利多大買 → 可能引發 [軋空](market-terms.md#軋空) |

---

## 借券（Securities Lending）

| 項目 | 說明 |
|------|------|
| **英文** | Securities lending |
| **定義** | 借入股票用於賣出或還券，與融券相關但統計口徑可能不同 |
| **在哪裡看到** | 借券賣出餘額 |
| **常見誤解** | 借券賣出等同看空；亦可能是法人避險或套利 |
| **小例子** | 借券賣出餘額攀升，但股價未跌 → 需綜合判斷 |

---

## 券資比（Short/Margin Ratio）

| 項目 | 說明 |
|------|------|
| **英文** | Short/margin ratio |
| **定義** | 融券餘額 / 融資餘額（或類似定義），衡量多空籌碼結構 |
| **在哪裡看到** | 個股籌碼面數據 |
| **常見誤解** | 數值愈高愈空；需看絕對餘額與產業慣例 |
| **小例子** | 券資比偏高且籌碼集中 → 軋空題材的觀察點之一 |

---

## 集保大戶（Major Holders, TDCC）

| 項目 | 說明 |
|------|------|
| **英文** | Major holders (TDCC concentration) |
| **定義** | 集保戶股權分散表，依持股張數分級，觀察大戶與散戶持股變化 |
| **在哪裡看到** | 週更新之集保資料 |
| **常見誤解** | 大戶增加一定利好；可能是佈局期，股價短期仍可能整理 |
| **小例子** | 400 張以上大戶持股比連續上升 → 籌碼趨於集中 |

---

## 期現價差（Basis） {#期現價差}

| 項目 | 說明 |
|------|------|
| **英文** | Basis (futures − spot) |
| **定義** | 台指期貨價格 − 加權指數現貨價格；正價差代表期貨高於現貨（情緒偏熱），逆價差代表期貨低於現貨（情緒偏保守） |
| **在哪裡看到** | 期貨報價對比加權指數 |
| **常見誤解** | 可精準預測當日漲跌；多作為**情緒參考**，非單一訊號 |
| **小例子** | 大幅逆價差 → 市場避險情緒升高的參考 |

---

## 買超 / 賣超（Net Buy / Net Sell） {#買超賣超}

| 項目 | 說明 |
|------|------|
| **英文** | Net buy / Net sell |
| **定義** | 買超：買進量（或金額）> 賣出，買方力道較強；賣超：賣出 > 買進，賣方力道較強 |
| **在哪裡看到** | 三大法人買賣超、分點進出、鉅額交易統計 |
| **常見誤解** | 買超一定漲；可能是當日短線，需看**連續多日**與股價 |
| **小例子** | 外資連 3 日買超 + 投信同向 → 籌碼面偏多參考 |

---

## 分點（Broker Branch） {#分點}

| 項目 | 說明 |
|------|------|
| **英文** | Broker branch |
| **定義** | 券商總公司底下的**營業據點**（各分行）；交易所每日公布各分點的買賣資料 |
| **在哪裡看到** | 分點進出排行、主力分點追蹤 |
| **常見誤解** | 某分點買就跟單；分點背後身分多元，難以保證獲利 |
| **小例子** | 特定分點連續布局某檔 → 進階籌碼研究的觀察點 |

**分點資料欄位**：

| 欄位 | 意義 |
|------|------|
| 分點名稱 | 哪一家券商哪一據點 |
| 買賣超 | 該分點當日淨買或淨賣 |
| 標的與張數 | 買賣哪檔、多少張 |

參考：[學吧十大名詞](../appendix/video-resources.md#學吧-learningpa-十大名詞) · [鉅額交易表](../03-tables/block-trade.md)

## 重點回顧

- 籌碼是「誰在買賣」，要與價格、基本面一起看。
- 融資融券反映槓桿與多空，不是單純多空指標。
- 法人資料 T+1 公布，有時間差。

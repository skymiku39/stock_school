# 風控術語

## 本篇你會學到

- 曝險、資金上限、閘門與門檻
- 緊急停機（Kill Switch）的白話解釋

---

## 曝險（Exposure） {#曝險}

| 項目 | 說明 |
|------|------|
| **英文** | Exposure |
| **定義** | 投入市場、可能因價格變動而損失的資金規模 |
| **在哪裡看到** | 交易計畫中的「單檔上限」「總資金上限」 |
| **常見誤解** | 只買 1 張就沒風險；股價跌 10% 仍可能虧數萬 |
| **小例子** | 總資金 50 萬，單檔最多 20 萬 → 單檔曝險 40% |

---

## 資金上限（Capital Cap）

| 項目 | 說明 |
|------|------|
| **英文** | Capital cap |
| **定義** | 當日或同時可投入市場的最大金額，避免過度集中 |
| **在哪裡看到** | 個人交易紀律、風控規則表 |
| **常見誤解** | 有錢就能滿倉；上限是為了控制單一時點的曝險 |
| **小例子** | 設定總曝險不超過資金 60% → 留現金應變 |

詳見 [資金配置](../06-risk/capital.md)。

---

## 閘門（Gate）

| 項目 | 說明 |
|------|------|
| **英文** | Gate (entry guard) |
| **定義** | 送單前必須通過的**檢查**；任一不通過就不應交易 |
| **在哪裡看到** | 交易前檢查清單、程式的進場守門 |
| **常見誤解** | 閘門是給程式用的；手動交易者也應有自己的「不交易清單」 |
| **小例子** | 今天已虧到每日上限、標的流動性不足、庫存對不起來 → 不開新倉 |

---

## 門檻（Threshold）

| 項目 | 說明 |
|------|------|
| **英文** | Threshold |
| **定義** | 觸發具體動作的**數值**，如停損 -3%、移動停利啟動 +6% |
| **在哪裡看到** | 停損停利設定、策略參數 |
| **常見誤解** | 門檻等同閘門；閘門是「能不能做」，門檻是「做到哪裡要動作」 |
| **小例子** | 淨利達 +6% → 啟動移動停利追蹤 |

---

## 緊急停機（Kill Switch） {#緊急停機kill-switch}

| 項目 | 說明 |
|------|------|
| **英文** | Kill switch |
| **定義** | 發現異常時**立即停止新進場**，但通常仍允許平倉既有部位 |
| **在哪裡看到** | 風控系統、交易紀律規則 |
| **常見誤解** | 停機 = 不賣；多數設計是「不買新的」，該平的倉仍要處理 |
| **小例子** | 程式重複下單或情緒失控 → 觸發停機強制冷靜 |

---

## 最大回撤（Maximum Drawdown）

| 項目 | 說明 |
|------|------|
| **英文** | Maximum drawdown |
| **定義** | 從資產高點到之後低點的最大跌幅百分比 |
| **在哪裡看到** | 策略績效報告、帳戶權益曲線；試算見 [組合管理](../09-advanced/portfolio.md#最大回撤試算) |
| **常見誤解** | 只看報酬率；回撤衡量最糟時期的痛苦程度 |
| **小例子** | 權益從 100 萬跌到 70 萬 → 最大回撤 30% |

---

## 風險報酬比（Risk-Reward Ratio）

| 項目 | 說明 |
|------|------|
| **英文** | Risk-reward ratio |
| **定義** | 預期獲利與可能虧損的比例，如賺 2 元願意賠 1 元 → 2:1 |
| **在哪裡看到** | 交易計畫、進場前評估 |
| **常見誤解** | 勝率高就夠；若贏小虧大，長期仍可能虧損 |
| **小例子** | 停損 -3%、目標 +6% → 風險報酬比 2:1 |

---

## 自我檢查

??? question "1.（概念題）風險報酬比 2:1 代表什麼？"
    參考答案：預期賺 2 單位願意賠 1 單位；勝率低仍可能長期虧損。

??? question "2.（判斷題）勝率高就一定賺錢？"
    參考答案：否。若**贏小虧大**，長期期望值仍可能為負。

??? question "3.（情境題）權益從 100 萬跌到 70 萬，最大回撤多少？"
    參考答案：**30%**；衡量最糟時期痛苦，見上表「最大回撤」小節。

## 重點回顧

- 風控先於選股：先定能虧多少，再談能賺多少。
- 閘門保護「不該交易的時候」；門檻保護「該出場的時候」。
- 緊急停機是紀律工具，不是失敗象徵。

# 如何選擇適合自己的模式

- 用五個問題篩選投資模式
- 各模式的門檻、成本、心理門檻與常見誤區
- 七種模式各自的 30 天學習計畫（週次任務）

| 頁面 | 適合你嗎 | 本篇定位 |
|------|----------|----------|
| **本篇** | 願意誠實回答五個問題、要門檻速查與 30 天計畫 | **問卷式篩選** |
| [對號入座](../10-persona/index.md) | 想依生活身分（上班族、學生…）快速對照 | **身分圖鑑** |

## 五個自問 {#五個自問}

| # | 問題 | 若答案是… | 較適合 |
|---|------|-----------|--------|
| 1 | 每天能專心盯盤多久？ | &lt; 30 分鐘 | 中長線、ETF、存股 |
| | | 1～4 小時 | 短線、隔日沖 |
| | | 盤中全時 | 當沖 |
| 2 | 能接受隔夜風險嗎？ | 不能 | 當沖 |
| | | 能，但怕跳空 | 短線為上限 |
| | | 能，看基本面 | 中長線 |
| 3 | 交易成本占獲利比重？ | 很小才可當沖 | 見 [交易成本](../06-risk/trading-costs.md) |
| 4 | 喜歡選股還是配置？ | 選股 | 個股各模式 |
| | | 配置 | [ETF](etf-investing.md) |
| 5 | 虧損時能紀律停損嗎？ | 常猶豫 | 先練 [中線](swing-mid.md) 或 ETF，勿當沖 |

!!! tip "誠實優於野心"
    許多虧損來自「用長線心態進場、用短線恐慌出場」或相反。選模式時以**你能執行的紀律**為準，不是以「哪種聽起來最賺」為準。

## 模式門檻速查

| 模式 | 資金門檻（概念） | 知識門檻 | 心理門檻 |
|------|------------------|----------|----------|
| 當沖 | 低（但成本占比高） | 分 K、量能、紀律 | 極高 |
| 隔日沖 | 低～中 | 日 K、跳空、夜盤 | 高 |
| 短線 | 中 | 技術+法人 | 中高 |
| 中線 | 中 | 營收+籌碼+技術 | 中 |
| 長期 | 中～高 | 基本面框架 | 中（需耐心） |
| 存股 | 中～高 | 除權息+財報 | 中 |
| ETF | 低 | 入門即可；0050 定額見 [專章](etf-passive-dca.md) | 低～中 |

### 心理門檻怎麼解讀

| 等級 | 意思 | 代表模式 |
|------|------|----------|
| **極高** | 盤中連續決策、停損零猶豫、輸贏當日消化 | 當沖 |
| **高** | 接受隔夜跳空、隔日果斷出場 | 隔日沖 |
| **中高** | 快砍小賠、不戀戰 | 短線 |
| **中** | 忍受數週橫盤、每週固定檢視 | 中線、長期、存股 |
| **低～中** | 閒錢、定額、少盯短線波動 | ETF |

若你自評「停損常猶豫」或「持股過夜就睡不著」，請**不要**從當沖或隔日沖開始。詳見 [心態錯配](mode-psychology.md#心態錯配)。

## 常見錯配 {#常見錯配}

| 錯配 | 後果 | 改正 |
|------|------|------|
| 沒時間盯盤卻當沖 | 錯過出場、違規留倉 | 改短線或中線 |
| 用當沖停損%做長線 | 被洗出好公司 | 改 [結構停損](../06-risk/stop-loss.md) |
| 存股卻每天看分 K | 情緒波動、亂加減碼 | 改週 K + 季報檢視 |
| 不看成本頻繁交易 | 賺小賠大 | 算 [損益平衡](../06-risk/trading-costs.md) |
| 長線標的、短線心態 | 洗盤低點被震出 | 對齊 [盯盤頻率](mode-psychology.md#選定模式後心態紀律三條) |
| 生活費投入股市 | 被迫低點變現 | [閒錢原則](../06-risk/capital.md#閒錢與生活費) |

## 重點回顧

- 用**時間、隔夜風險、成本、選股vs配置、紀律**五題篩選。
- 選定一種主模式後，至少 30 天不要跳別種模式。
- 對照 [投資模式與心態](mode-psychology.md) 調整盯盤頻率與賣出條件。
- 回 [投資模式總覽](index.md) 進入對應專章。

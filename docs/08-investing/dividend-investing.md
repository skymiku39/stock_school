## 參與除權息流程

```mermaid
flowchart LR
    A[選穩健標的] --> B[確認最後過戶日]
    B --> C[除息參考價下修]
    C --> D[觀察填息與法人]
    D --> E[長期持有或滾存股利]
```

| 時點 | 注意 |
|------|------|
| 除息前 | 勿盲目「搶息」追高 |
| 除息日 | 股價下修為正常，非帳面虧損 |
| 除息後 | 填息需基本面支撐 |

完整案例：[除權息參與與填息](../07-cases/dividend-play.md)

## 成本與稅

- [交易成本](../06-risk/trading-costs.md)：存股頻率低，成本占比小。
- 股利所得、二代健保等依個人狀況，以稅務專業為準。
- [分批](../02-glossary/trading-terms.md#分批) 佈局可降低除息前追高風險。

## 與其他模式

| 可並行 | 避免 |
|--------|------|
| 長期價值（成長+配息） | 用分 K 每天交易存股標的 |
| ETF 高股息 | 把存股當當沖標的 |

## 心態與建議

| 面向 | 存股除權息 |
|------|------------|
| 心理關鍵 | 除息下修是流程；重配息品質非殖利率數字 |
| 常見陷阱 | 除息前搶息追高、把配息當定存保證 |
| 盯盤 | 除權息日程 + 季報；少看分 K |
| 延伸 | [存股心態詳解](mode-psychology.md#存股心態) |

## 自我檢查

??? question "1.（概念題）存股該看殖利率還是配息品質？"
    參考答案：看**配息品質與永續性**（獲利、現金流支撐），高殖利率可能是股價跌出來的假象。

??? question "2.（判斷題）除息前搶進賺股利是穩賺的嗎？"
    參考答案：不是。股價已反映股利，還要看**填息**與股利稅、二代健保，見 [除權息誤操作案例](../07-cases/ex-dividend-mistake.md)。

??? question "3.（情境題）存股標的某年配息明顯減少，你會怎麼查？"
    參考答案：查 [財報的現金流與配息永續性](../03-tables/financials.md#配息永續性)，判斷是一次性還是長期惡化。

## 重點回顧

- 存股看**配息品質**，不是只看殖利率數字。
- 填息是結果，需營收與籌碼配合。

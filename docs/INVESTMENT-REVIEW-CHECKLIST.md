# 投資專業審查檢核表（維護者）

本頁是 Stock School 教材的**投資專業驗收依據**，供維護者在新增章節、潤稿或定期複審時，以資深台股投資人視角逐項檢查內容是否正確、用語是否專業、風險是否揭露。學員可略過本頁。

!!! note "定位"
    本頁規範「投資內容對不對」；「怎麼寫」見 [寫作規範](STYLE-GUIDE.md)，「圖表完整性」見 [圖表檢查清單](CHART-CHECKLIST.md)，內容權威來源見 [架構說明](ARCHITECTURE.md)。完整一輪審查結果記錄於 `reports/investment-review.md`。

---

## 五個審查維度

| 維度 | 檢查重點 |
|------|----------|
| **事實正確性** | 稅率、交割、漲跌停、除權息參考價、ETF 費用、融資融券與保證金規則是否與台股實務一致 |
| **專業用語** | 是否符合業界慣用（PER／本益比、軋空、填息vs填權、MoM vs QoQ、thesis）；對照 [寫作規範](STYLE-GUIDE.md) 與 [完整詞條](02-glossary/dictionary.md) |
| **概念框架** | 期望值、風險報酬比、核心衛星、技術指標解讀是否邏輯自洽、無因果倒置 |
| **風險揭露** | 是否標示「教學用」、常見誤區、強勢股例外（如 RSI 超買續漲）；對照 [免責聲明](appendix/disclaimer.md) |
| **教學取捨** | 白話簡化屬「可接受簡化」或會造成實戰誤判的「有害簡化」 |

## 嚴重度分級

| 等級 | 定義 | 處置 |
|------|------|------|
| **P0 致命** | 可能導致虧損或違規認知（錯誤稅率、錯誤交割、暗示無風險套利） | 立即修正 |
| **P1 重大** | 概念錯誤或術語誤用，影響決策品質 | 當批修正 |
| **P2 輕微** | 表述不精確、數字不一致、與 canonical 章節不一致、繁簡混用 | 修正或加 cross-link |
| **P3 建議** | 可讀性、資深視角補充（nuance、反例） | 視篇幅納入 admonition |

## 提交前逐項檢查

### A. 數字與事實（每次改動含數字的頁面）

- [ ] 稅費引用與 [交易成本](06-risk/trading-costs.md) 一致：個股 **0.3%**、ETF **0.1%**、現股當沖 **0.15%**（優惠稅率，注意時效）、手續費 **0.1425%**。
- [ ] 交割為 **T+2**；漲跌停 **±10%**；除息參考價 ≈ 昨收 − 現金股利。
- [ ] 期貨保證金、每點價值（大台 200／小台 50／微台 10 元）標明「隨期交所公告調整」，且槓桿以倍數呈現而非僅絕對金額。
- [ ] 自製試算（複利、損益平衡、期望值）可重算驗證，且各列內部一致。
- [ ] 指標參數符合慣例：RSI 14、MACD 12,26,9、KD 9,3,3、布林 20,2。

### B. 專業用語

- [ ] MoM（月對月）、QoQ（季對季）、YoY（年對年）依資料頻率正確使用。
- [ ] 軋空、填息、填權、缺口、利多出盡等用語與 [市場進階術語](02-glossary/market-terms.md) 定義一致。
- [ ] 無簡體字（如 这／实／轨／领／侥幸）；英文術語僅首次括註。

### C. 概念與風險

- [ ] 技術指標頁標明**滯後性**與**強勢股例外**（超買可續漲、低檔可續跌）。
- [ ] 籌碼定位為**確認層**，不取代基本面 thesis；強調看連續而非單日。
- [ ] 估值需**同業、歷史、成長**三維比較，不給「PER 15 就便宜」的萬用規則。
- [ ] ETF／定期定額不暗示「長期一定賺」「跟定存一樣」。

### D. 案例專屬

- [ ] 遵循模板：背景 → 表／圖 → 推理 → 結論（教學用）→ 反思。
- [ ] 結論段含「不建議／非投資建議」，無隱含買賣信號。
- [ ] 含反例或「警訊不是賣出指令」類 nuance，並附免責聲明。

## 定期複審節奏

| 頻率 | 動作 |
|------|------|
| 每次潤稿 | 跑 A、B 區對應頁面 |
| 季度 | 重新核對稅率、保證金、優惠稅率時效是否變更（法規可能調整） |
| 重大法規變動時 | 全站搜尋受影響數字並更新，於 commit 說明理由 |

法規時效以主管機關（證交所、櫃買、財政部、金管會、期交所）最新公告為準；爭議項標註「請以主管機關公告為準」。

## 重點回顧

- 投資審查與工程 CI（pytest、連結稽核、strict build）**互補**：CI 保證連結與資產完整，本表保證投資內容正確。
- 最高優先永遠是 **P0**：任何可能讓讀者賠錢或誤判合規的敘述。
- 修正以**最小 diff**、維持白話、用 admonition 補 nuance 為原則。

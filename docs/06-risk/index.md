# 風險與紀律總覽

## 本篇你會學到

- 為什麼**活下來**比一時獲利更重要
- 資金、成本、停損、紀律四層的關係
- 建議的閱讀順序

!!! warning "免責聲明"
    本章為風險管理觀念整理，不構成投資建議。實際操作請依個人財務狀況評估。


| 層級 | 回答的問題 | 專章 |

---

## 建議閱讀順序

1. [資金配置](capital.md)：先決定能虧多少、怎麼分配。
2. [交易成本與期望值](trading-costs.md)：算清楚每筆交易的損益平衡點。
3. [停損三層概念](stop-loss.md)：設定出場規則。
4. [交易紀律](discipline.md)：把規則變成習慣。

## 重點回顧

- 風控的目標是**留在場上**，而非每筆都賺。
- 資金 → 成本 → 停損 → 紀律，四層缺一不可。
- 下一步：用 [實戰案例](../07-cases/index.md) 看風控如何影響結果。

# 交易成本與期望值

## 本篇你會學到

- 手續費與證交稅如何吃掉獲利
- 損益平衡點與期望值概念

## 費用結構（台股現股） {#費用結構}

| 項目 | 費率（概略） | 說明 |
|------|-------------|------|
| 買入手續費 | 0.1425%（常打折） | 券商折扣常見 28 折 ≈ 0.04% |
| 賣出手續費 | 同上 | 賣出時再收一次 |
| 證交稅（個股現股） | **0.3%** | **僅賣出**時課徵 |
| 證交稅（ETF 賣出） | **0.1%** | 低於個股；買賣手續費機制同個股 |
| 證交稅（現股當沖） | **0.15%**（概略） | 須符合當沖資格；見 [市場概覽](../01-basics/market-overview.md#當沖當日沖銷) |

交割時點（T+2）與「買進當下扣什麼」→ [交割與稅費](../01-basics/settlement-fees.md)。

ETF **內扣管理費**、折溢價、三層費用圖 → **[ETF 費用與折溢價](../01-basics/etf-costs-and-premium.md)**（不在此重複展開）。

實際費率以券商與當時法規為準。

## 損益平衡點（當沖範例）

假設手續費 28 折（約 0.04% 單邊）+ 當沖證交稅 0.15%：

```
損益平衡漲幅 ≈ 0.04% + 0.04% + 0.15% ≈ 0.23%
```

股價必須漲超過約 **0.23%**（淨利概念），當沖才算真的賺到錢。

| 毛利 | 結果 |
|------|------|
| +0.15% | 可能仍虧（扣費後） |
| +0.5% | 小賺 |
| +2% | 較明顯獲利（若停損紀律良好） |

## 一筆計算範例

買進 100 元 × 2 張 = 200,000 元；賣出 103 元。

| 項目 | 金額（示意） |
|------|-------------|
| 毛利 | 6,000 |
| 買賣手續費 | 約 160 |
| 證交稅（當沖） | 約 309 |
| **淨利** | 約 5,530 |

短線「賺 1%」有時扣完只剩微薄淨利——這是當沖必須嚴守 [停損](stop-loss.md) 的原因之一。

## 期望值

```
期望值 = 勝率 × 平均獲利 − (1 − 勝率) × 平均虧損
```

| 情境 | 勝率 | 平均賺 | 平均賠 | 期望值 |
|------|------|--------|--------|--------|
| A | 40% | 2% | 1% | 0.8% − 0.6% = **+0.2%** |
| B | 50% | 1% | 1% | **0%**（白忙） |
| C | 60% | 1% | 2% | 0.6% − 0.8% = **−0.2%** |

!!! warning "贏多輸少才夠"
    勝率 60% 若平均賺小、平均賠大，長期仍可能虧損。需同時看**盈虧比**。

## 與資金配置的關係

單筆最大虧損應先算清楚（見 [資金配置](capital.md)），再評估每月交易次數與成本：

```
月成本 ≈ 交易次數 × 單筆來回費用
```

過度交易會讓期望值被成本吃掉。

## 重點回顧

- 當沖先看損益平衡點（約 0.2%～0.3% 量級，視折扣而定）。
- 勝率與盈虧比要一起算期望值。

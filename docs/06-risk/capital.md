# 資金配置

## 本篇你會學到

- 保守 / 標準 / 積極三種資金分級
- 單筆最大虧損怎麼算

## 資金分級（教學參考）

| 等級 | 總資金範例 | 單檔張數建議 | 適用 |
|------|------------|--------------|------|
| **保守** | 20 萬 | 1 張 | 初期學習、驗證紀律 |
| **標準** | 50 萬 | 1～2 張 | 策略穩定後 |
| **積極** | 100 萬+ | 2～3 張 | 有經驗且紀律嚴格 |

!!! warning "提醒"
    以上為教學示意，請依個人財務狀況調整，勿動用生活必需資金。

## 閒錢與生活費 {#閒錢與生活費}

| 類型 | 說明 |
|------|------|
| **閒錢** | 短時間內不會動用；不含生活費、房租、學費、**勞健保等固定扣款**、緊急預備金 |
| **不該投入股市的錢** | 近期會用到的資金、借來的錢 |

算閒錢時別只看帳戶餘額——**勞保、全民健保費**等每月固定支出要先留好；**勞退新制**、**國債／定存／債券 ETF** 等屬其他資產桶，與台股分開評估（提醒見 [組合管理](../09-advanced/portfolio.md#個人總資產的其他桶)）。

### 認賠殺出 {#認賠殺出}

若用生活費或短期會用到的錢投資，突發狀況急需變現時若剛好碰上低迷，容易被迫在**最差時機**賣出 — 俗稱**認賠殺出**，虧損從帳面變成實際損失。

定期定額、長期持有（如 [0050 專章](../08-investing/etf-passive-dca.md#為什麼強調閒錢)）的前提是：你有能力**撐過低谷**，不必為現金流在低點砍倉。

## 分配原則

| 原則 | 說明 |
|------|------|
| 單檔上限 | 單一標的不超過總資金 30%～40% |
| 同時持倉數 | 當沖宜少檔；中線可分散 3～5 檔 |
| 保留現金 | 預留應對滑價、追加保證金、情緒冷靜 |

## 單筆最大虧損

```
單筆最大虧損 ≈ 股價 × 1,000 × 張數 × |停損淨利%|
```

**範例**：股價 500 元、2 張、停損淨利 -3%

```
500 × 1,000 × 2 × 0.03 = 30,000 元
```

這代表：若停損紀律確實執行，這筆交易最多虧約 3 萬（未含極端滑價）。

## 每日風險預算 {#每日風險預算}

| 做法 | 說明 |
|------|------|
| 每日最大虧損 | 如總資金 1%～2%，達到即停止開新倉 |
| 連續虧損停手 | 連續 3 筆停損 → 當日結束 |

## 當沖與交割

即使當日沖銷不留倉，帳戶仍須具備**足夠交割能力**。詳見 [交易流程](../01-basics/trading-flow.md)。

## 自我檢查

??? question "1.（概念題）「閒錢」為什麼不能包含下個月房租？"
    參考答案：短期會動用的錢被迫在低點變現時，容易 [認賠殺出](#閒錢與生活費)；應先扣固定支出與緊急預備金。

??? question "2.（判斷題）總資金 50 萬，單檔買 3 張高價股沒問題？"
    參考答案：要先算單檔是否超過總資金 30%～40% 上限，並用停損% 反推「最慘能虧多少」。

??? question "3.（情境題）股價 500、2 張、停損淨利 -3%，單筆最大虧損約多少？"
    參考答案：500 × 1,000 × 2 × 0.03 = **30,000 元**（見上方公式）。

## 重點回顧

- 先算「最慘能虧多少」，再決定張數。
- 資金分級是紀律起點，不是獲利保證。

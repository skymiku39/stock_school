# 除權息入門

## 本篇你會學到

- 除權、除息、填息的意義
- 除權息對股價與殖利率的影響

## 名詞

| 名詞 | 說明 |
|------|------|
| **現金股利（息）** | 公司配發現金給股東 |
| **股票股利（權）** | 公司配發股票（免費股數增加） |
| **除息日** | 這天起買進不再享有本次現金股利；股價會依股利調整 |
| **除權日** | 這天起買進不再享有本次股票股利；股價會依權值調整 |
| **填息 / 填權** | 除權息後股價回漲至除權息前水準 |

## 股價為什麼會「跳」

除息簡化概念：若配息 3 元，除息日開盤參考價約為昨收 **減 3 元**（實際以交易所計算為準）。

!!! note "注意"
    除權息不是「免費送錢」——股價會相應調整，總資產在除權息當下應大致不變（未計稅）。

## 殖利率

```
殖利率 ≈ 現金股利 / 股價 × 100%
```

| 現象 | 解讀 |
|------|------|
| 股價大跌 | 殖利率被動拉高 → 可能是景氣或獲利問題 |
| 穩定配息 + 股價平穩 | 存股族常關注的組合 |

詳見 [估值表](../03-tables/valuation.md) 與 [基本面術語](../02-glossary/fundamentals.md#殖利率)。

## 參與除權息

| 規則（簡化） | 說明 |
|--------------|------|
| 股東會公告 | 在除權息 **前** 買進並持有過戶日，才能領股利 |
| 最後過戶日 | 見券商或公開資訊觀測站公告 |

## 重點回顧

- 除權息是權利分配，股價會調整，不是無風險套利。
- 高殖利率需搭配獲利穩定性，見 [估值陷阱案例](../07-cases/valuation-trap.md)。

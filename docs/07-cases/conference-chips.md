# 案例七：法說樂觀 vs 法人賣超

## 本篇你會學到

- 法說敘事與**法人實際動向**出現落差時怎麼看
- 中線投資論點（thesis）如何檢視與修正
- 適用模式：[中線波段](../08-investing/swing-mid.md)

!!! warning "免責聲明"
    匿名教學案例，非投資建議。

## 背景

「G 公司」法說會管理層表示「下半年展望樂觀、訂單能見度佳」。媒體標題偏多。

## 看到的資料

| 來源 | 內容 |
|------|------|
| 法說摘要 | 語氣樂觀，但未給具體營收數字 |
| 三大法人 | 法說後 5 日合計賣超 3,000 張 |
| 股價 | 法說當日漲 3%，後 3 日回落 |
| 融資 | 散戶融資餘額增加 |

## 推理步驟（言行反查）

1. **話術檢查**：「樂觀」是否量化？有無下修風險揭露？見 [法說重點](../05-analysis/conference.md)。
2. **籌碼行為**：法人連賣與樂觀說法**不一致**。
3. **股價反應**：利多出盡式上漲後回落。
4. **融資增**：散戶追高跡象，非機構跟進。
5. **結論**：提高警覺，不因標題買進；等營收或法人方向確認。

## 何時算「一致」

| 現象 | 解讀 |
|------|------|
| 樂觀法說 + 法人買超 + 價穩 | 較一致 |
| 樂觀法說 + 法人賣超 | 言行反查，需深入 |

## 重點回顧

- 標題新聞 < 原始法說 < 籌碼行為。

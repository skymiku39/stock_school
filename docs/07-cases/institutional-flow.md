# 案例三：三大法人連續買超

## 本篇你會學到

- 法人**連續買超**如何搭配個股趨勢
- 籌碼面在短中線的權重
- 適用模式：[短線](../08-investing/swing-short.md) · [中線](../08-investing/swing-mid.md)

!!! warning "免責聲明"
    匿名教學案例，非投資建議。

## 背景

「C 公司」為中型電子股，股價在 80～90 元區間盤整兩週。你想知道法人籌碼是否支持突破。

## 看到的表

| 日期 | 外資(張) | 投信 | 合計 | 收盤 |
|------|---------:|-----:|-----:|-----:|
| D-4 | +500 | +50 | +550 | 84 |
| D-3 | +300 | +80 | +380 | 85 |
| D-2 | +200 | +40 | +240 | 86 |
| D-1 | +600 | +100 | +700 | 87 |
| D0 | +400 | +60 | +460 | 88 |

5 日外資累計約 +2,000 張；股價緩步上移。

## 推理步驟

1. **連續性**：5 日合計皆買超，非單日異常 → 符合 [法人表](../03-tables/institutional.md) 的「連續買超」讀法。
2. **價量配合**：股價緩漲、非暴衝 → 較像佈局而非追高末段。
3. **融資**：同期融資餘額微增、未暴增 → 散戶槓桿未過熱（見 [融資表](../03-tables/margin.md)）。
4. **壓力區**：90 元為前波密集區 → 突破需放量。
5. **反例準備**：若突破 90 失敗且法人轉連賣 → 假突破。

## 反例（同一邏輯的失敗故事）

| 現象 | 解讀 |
|------|------|
| 連續買超但股價跌 | 可能有更大賣壓（如大股東釋出） |
| 單日超大買超後消失 | 可能是指數調整或被動基金 |
| 買超在歷史高檔 | 可能是短線過熱 |

## 結論（教學用）

- **偏多觀察**：低檔盤整 + 法人連買 + 股價緩漲 → 籌碼面支持突破測試。
- **觸發**：收盤站上 90 且量增，可視為短線進場參考；停損設 85 下方。
- **失效**：連續 2 日合計賣超且跌破 84 → 籌碼故事弱化。

## 重點回顧

- 法人要看**連續性**與**價格是否配合**。
- 準備反例，避免「法人買超必漲」迷思。

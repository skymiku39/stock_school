    行情與報價¶

    本篇你會學到¶

    • 開高低收、成交量、漲跌幅的意義
    • 委買委賣與漲跌停
    • 上市、上櫃在哪裡交易、散戶該注意什麼

    開高低收（OHLC）¶

    項目 說明
    英文 OHLC（Open, High, Low, Close）
    定義 一根 K 棒的四個價格：開盤（該時段第一筆成交價）、最高、最低、收盤（最後一筆成交價）
    在哪裡看到 K 線圖、個股報價頁、日行情表；2330／0050 範例見 報價畫面
    常見誤解 紅K = 漲、黑K = 跌；「紅K」只代表收盤 ≥ 開盤，不一定相對昨收上漲
    小例子 開 100、高 105、低 99、收 104 → 紅K，實體從 100 到 104

    漲跌幅（Change %）¶

    項目 說明
    英文 Change %
    定義 (現價或收盤價 − 昨收) / 昨收 × 100%
    在哪裡看到 報價列表第一欄、新聞標題「漲 X%」
    常見誤解 漲 5% 不代表「還會再漲 5%」；只是相對昨收的變化
    小例子 昨收 200，今收 210 → 漲跌幅 +5%

    成交量（Volume）¶

    項目 說明
    英文 Volume
    定義 該時段成交的股數（常見以張或千股顯示）
    在哪裡看到 K 線下方柱狀圖、個股總覽表的「量」欄
    常見誤解 量大一定好；可能是換手、出貨或主力對倒，需搭配價格方向
    小例子 突破月線時放量 → 較有說服力；跌停放量 → 可能是恐慌殺出

    委買 / 委賣（Bid / Ask）¶

    項目 說明
    英文 Bid / Ask
    定義 尚未成交的掛單價格與張數；委買為買方掛單，委賣為賣方掛單
    在哪裡看到 五檔或十檔報價；圖解見 報價畫面
    常見誤解 委買很多就會漲；大單可能撤單，且成交才是結果
    小例子 委買五檔張數遠大於委賣 → 短線買盤較積極（仍可能變化）

    漲跌停（Limit Up / Down）¶

    項目 說明
    英文 Limit up / Limit down
    定義 個股當日價格相對昨收的上下限（常見 ±10%）
    在哪裡看到 報價顯示「漲停」「跌停」、一字線 K 棒
    常見誤解 漲停要追；追漲停風險高，且次日不一定續漲
    小例子 昨收 100，漲停價約 110、跌停價約 90

    昨收（Previous Close）¶

    項目 說明
    英文 Previous close
    定義 前一交易日最後成交價，為當日漲跌幅計算基準
    在哪裡看到 報價頁「參考價」「昨收」
    常見誤解 昨收等於開盤；開盤會因隔夜消息跳空高於或低於昨收
    小例子 昨收 100、今開 102 → 開高 2 元

    平盤與開高、開低（Flat / Gap-up / Gap-down Open）¶

    項目 說明
    英文 Flat / Gap-up open / Gap-down open
    定義 平盤：口語常指昨收價，亦作當日競價與漲跌幅基準；開高：開盤 > 昨收；開低：開盤 < 昨收
    在哪裡看到 開盤瞬間報價、分時圖起點
    常見誤解 開高一定會漲到底；開高後仍可能一路走低收黑K
    小例子 開高走低：開盤偏高但盤中下跌；開低走高：開盤偏低但盤中上漲

    參考影片：學吧十大名詞

    前高、前低（Prior High / Prior Low）¶

    項目 說明
    英文 Prior high / Prior low
    定義 前高：目前價位之前最近的一個相對高點；前低：之前最近的一個相對低點
    在哪裡看到 日 K、分 K（江波圖）、畫線支撐壓力
    常見誤解 突破前高必續漲；需搭配成交量確認
    小例子 突破前高常視為多方延續；跌破前低則偏空

    相關：支撐與壓力 · 回測支撐

    上市（TWSE Listed）¶

    項目 說明
    英文 Listed on TWSE
    定義 股票在台灣證券交易所掛牌買賣；門檻較高，常見大型權值股、ETF，成交量通常較大
    在哪裡看到 個股頁標示「上市」、加權指數成分、Yahoo 股市市場欄；三層比較見 市場概覽
    常見誤解 上市就一定安全或好賺；仍要看個股基本面、流動性與風控
    小例子 2330 台積電、0050 為上市標的，盤中 09:00–13:30 於證交所撮合

    上櫃（TPEx OTC）¶

    項目 說明
    英文 OTC listed on TPEx
    定義 股票在證券櫃檯買賣中心（櫃買中心）掛牌；常見中小型、生技、成長型公司，流動性個股差異大
    在哪裡看到 個股頁標示「上櫃」、櫃買指數（OTC）、TPEx 官網行情
    常見誤解 上櫃 = 品質差；許多優質公司選擇上櫃，但小型上櫃股成交量可能偏低，買賣價差與滑價風險較高
    小例子 同代號 4 碼個股，在 Yahoo 股市可見「上市」或「上櫃」標籤；上櫃小型股宜先查 成交量 再下單

    上櫃散戶實務

    上櫃不是不能買，但要更注意：成交量是否夠你進出、停損是否設好、是否被列 注意股／處置股。與上市差異總表見 市場概覽。

    自我檢查¶

    1.（概念題）台股紅綠色與美股有何不同？

    參考答案：台股紅漲綠跌；美股常為綠漲紅跌。

    2.（判斷題）五檔報價等於已成交價格？

    參考答案：否。五檔是掛單意願，不是成交結果。

    3.（情境題）上櫃小型股成交量每天不到 100 張，下單前該注意什麼？

    參考答案：流動性可能不足、買賣價差大；先查 成交量過濾 與 上櫃 說明。

    4.（手算題）昨收 50、現價 52.5，漲跌幅多少？

    參考答案：(52.5−50)÷50×100% = 5%。

    重點回顧¶

    • 開高低收描述一根 K 棒的價格範圍；漲跌幅是相對昨收。
    • 成交量要與價格方向一起看。
    • 漲跌停是當日限制，不是趨勢預測。
    • 上市在證交所、上櫃在櫃買中心；上櫃小型股要特別注意流動性。

## 審查維度與嚴重度

| 維度 | 說明 |
|------|------|
| 事實正確性 | 稅率、交割、漲跌停、除權息參考價、ETF 費用、融資融券規則是否與台股實務一致 |

| 概念框架 | 期望值、風險報酬比、核心衛星、技術指標解讀是否邏輯自洽、無因果倒置 |
| 風險揭露 | 是否足夠標示教學用、常見誤區、強勢股例外 |
| 教學取捨 | 白話簡化屬「可接受簡化」或「有害簡化」 |

| 等級 | 定義 | 處置 |
|------|------|------|
| P0 致命 | 可能導致虧損或違規認知（錯誤稅率、錯誤交割、暗示無風險套利） | 立即修正 |
| P1 重大 | 概念錯誤或術語誤用，影響決策品質 | 本階段修正 |
| P2 輕微 | 表述不精確、缺脈絡、與 canonical 章節不一致 | 修正或加 cross-link |
| P3 建議 | 可讀性、資深視角補充（nuance、反例） | 視篇幅納入 admonition |
